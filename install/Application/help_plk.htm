Korzystanie z programu Data Replicator 3

Kliknij następujące główne funkcje, aby zapoznać się z programem Data Replicator 3.

  • Kopia zapasowa danych komputera
  • Przywracanie plików kopii zapasowej
  • Ustawienia zaawansowane
  • Log

Kopia zapasowa danych komputera

Wykonaj następujące kroki, aby skonfigurować kopię zapasową. Możesz też przejść do sekcji Ustawienia zaawansowane, aby dowiedzieć się o innych ustawieniach, które pomogą Ci spersonalizować tworzenie kopii zapasowej danych komputera.

  1. Wybierz miejsce docelowe kopii zapasowej

    2. Możesz wybrać tworzenie kopii zapasowej danych z komputera na serwerze Synology albo w innej lokalizacji. Po udanym wykonaniu kopii zapasowej pliki kopii zapasowej będą się znajdować w przypisanym folderze.

    1.1 Kopia zapasowa na serwerze Synology

    1. Kliknij Wybierz.
    2. WybierzSerwer Synology, a następnie kliknij OK.
    3. W kolejnym oknie kliknij serwer Synology, na którym znajduje się folder współdzielony na kopię zapasową. Następnie kliknij Dalej.
    4. Wprowadź nazwę użytkownika i hasło z uprawnieniami do Odczytu/Zapisu w wybranym folderze współdzielonym i kliknij Dalej.
    5. Kliknij folder współdzielony, którego chcesz użyć do tworzenia kopii zapasowej, i kliknij Zakończ.

    1.2 Kopia zapasowa w innej lokalizacji

    1. Kliknij Wybierz.
    2. Wybierz Inna lokalizacja i kliknij Przeglądaj, aby zlokalizować miejsce docelowe kopii zapasowej.
    3. Kliknij OK, aby zakończyć.

  2. Zaznacz foldery do uwzględnienia w kopii zapasowej

  3. Wybierz tryb kopii zapasowej

    4. Data Replicator 3 oferuje 3 tryby kopii zapasowej: Natychmiastowa, Synchronizacja i Harmonogram.

    3.1 Natychmiastowa

         Po kliknięciu przycisku Natychmiastowa wszystkie pliki w wybranych folderach lokalnych zostaną natychmiast skopiowane do miejsca docelowego kopii zapasowej.

    3.2 Synchronizacja

    • Zsynchronizuj dane teraz i rozpocznij monitorowanie

      • Kliknij przycisk Synchronizacja. Data Replicator 3 wykona pełną kopię zapasową wszystkich plików w wybranych folderach w miejscu docelowym kopii zapasowej. Następnie Data Replicator 3 zacznie monitorować te foldery. Po dokonaniu zmian Data Replicator 3 skopiuje tylko zmienione pliki.

      Uwaga:

      1. Po przejściu programu Data Replicator 3 do trybu monitoringu będzie on pracował aż do zatrzymania.
      2. Interwał między synchronizacjami poczty oznacza czas między zakończeniem jednej synchronizacji a rozpoczęciem kolejnej. Zadanie może zostać opóźnione, kiedy Data Replicator 3 przeprowadza synchronizację innych plików.

    • Zatrzymaj monitorowanie

      • Aby zatrzymać monitorowanie, kliknij przycisk Zatrzymaj synchronizację w oknie programu Data Replicator 3. Jeśli okno programu zostało zminimalizowane do zasobnika systemowego, kliknij dwukrotnie ikonę Data Replicator 3.

    3.3 Harmonogram

    • Ustaw harmonogram

      • Program Data Replicator 3 można ustawić tak, aby regularnie wykonywał automatyczne kopie zapasowe o określonej porze. Aby to zrobić:

      1. Kliknij Harmonogram.
      2. Ustaw okres kopii zapasowej i czas rozpoczęcia dla zaplanowanej kopii zapasowej i kliknij OK.
      3. Jeśli chcesz tworzyć zaplanowaną kopię zapasową podczas wylogowywania, wprowadź nazwę użytkownika i hasło, których używasz podczas logowania się do tego komputera. Następnie kliknij OK.
        Jeśli nie chcesz tworzyć zaplanowanej kopii zapasowej podczas wylogowywania, kliknij Pomiń.

      Uwaga:

      1. Aby można było wykonać kopię zapasową podczas wylogowania, hasło użytkownika nie może być puste.
      2. W systemach Windows 98 lub Windows ME zaplanowana kopia zapasowa może z założenia być wykonywana podczas wylogowywania.

      Po ustawieniu harmonogramu możesz wyjść z programu Synology Data Replicator 3 lub zminimalizować okno programu do zasobnika systemowego. Kiedy nadejdzie czas, Synology Data Replicator 3 automatycznie wykona zaplanowaną kopię zapasową.

    • Usuń plan

      • Kliknij Usuń plan, aby zatrzymać zaplanowaną kopię zapasową i usunąć ustawienia harmonogramu.

    Jeśli chcesz zapisać kopię zapasową plików programów Outlook, Outlook Express lub Windows Mail, zwróć uwagę, że:

    • Jeśli używasz systemów Windows XP, Windows 2003 lub Windows Vista
      1. Przynajmniej jeden z napędów twojego komputera powinien być w formacie NTFS.
      2. Pliki programów Outlook/Windows Mail powinny być na dyskach lokalnych.
      3. Musisz się zalogować jako administrator komputera.
    • Jeśli używasz innych systemów operacyjnych niż Windows XP i Windows 2003, zamknij program Outlook przed utworzeniem kopii zapasowej jego plików.

      • Obsługa i ograniczenie

      1. Szukając serwera docelowego, Synology Data Replicator 3 użyje portu 9999, 9998 lub 9997.
      2. Po wykonaniu kopii zapasowej skompresowane pliki i pliki zaszyfrowane staną się zwykłymi plikami. Właściwości plików zostaną skasowane.
      3. Synology Data Replicator 3 obsługuje nazwy plików o długości do 240 bajtów.
      4. Jeśli zmienisz nazwę swojego komputera PC, musisz wprowadzić hasło logowania administratora podczas ustawiania zaplanowanej kopii zapasowej, aby mieć pewność, że Data Replicator 3 będzie działać prawidłowo. Jeśli już został ustawiony harmonogram kopii zapasowej, usuń go i użyj hasła logowania administratora komputera, aby ustawić nowy.

Przywracanie plików kopii zapasowej

Aby przywrócić pliki kopii zapasowej, musisz najpierw wybrać miejsce docelowe kopii zapasowej, a następnie wybrać jedną z następujących metod przywracania plików kopii zapasowej:

  1. Przywróć przeglądając

    2. W większości sytuacji, kiedy masz pewność, gdzie znajdują się pliki kopii zapasowej, możesz użyć funkcji Przywróć przeglądając, aby przywrócić swoje pliki:

    1. Kliknij przycisk Uruchom pod Przywróć przeglądając.
    2. Kliknij Dalej.
    3. Kliknij, aby wybrać źródło przywracania, a następnie kliknij Dalej.
    4. Wybierz wyświetlanie plików Według wersji lub Według punktu przywracania.
    5. Zaznacz foldery do przywrócenia, a następnie kliknij Dalej.
    6. Wybierz lokalizację do przywrócenia kopii zapasowej plików. Następnie kliknij Zakończ.

  2. Przywróć wyszukując

    3. Jeśli nie masz pewności, gdzie znajdują się pliki kopii zapasowej, albo jeśli chcesz przywrócić tylko pliki z określonym słowem kluczowym, wykonaj następujące kroki:

    1. Kliknij przycisk Uruchom pod Przywróć wyszukując.
    2. Kliknij Dalej.
    3. Kliknij, aby wybrać źródło przywracania, a następnie kliknij Dalej.
    4. Wpisz słowo kluczowe, które chcesz wyszukać w polu tekstowym „Słowo Kluczowe:”.
    5. Wybierz folder kopii zapasowej z menu rozwijanego „Kopie zapasowe:”.
    6. Kliknij Szukaj.
    7. Wybierz pliki i foldery do przywrócenia z ramki po prawej stronie, a następnie kliknij Dalej.
    8. Wybierz lokalizację do przywrócenia kopii zapasowej plików. Następnie kliknij Zakończ.

    Porada dotycząca przywracania plików programów Outlook, Outlook Express oraz Windows Mail

         Podczas wybierania ścieżki docelowej możesz wybrać następujące opcje:

    1. Ścieżka oryginalna
      2.

      Wybierając tę opcję, nie musisz ręcznie importować poczty i książki adresowej. Dane z komputera zostaną zastąpione przez przywrócone pliki.

    2. Nowa ścieżka
      3.

      Wybierając tę opcję, możesz przywrócić dane do określonej ścieżki, a następnie ręcznie importować pocztę, książkę adresową i pocztę programów Outlook/Outlook Express/Windows Mail.

Ustawienia zaawansowane

W sekcji Opcje możesz skonfigurować następujące zaawansowane ustawienia kopii zapasowej.

Filtruj

Użyj tej funkcji, aby precyzyjnie kopiować pliki. Wykluczenie z listy kopii zapasowej plików z określonym prefiksem lub rozszerzeniem umożliwi szybsze tworzenie kopii zapasowej. Wykonaj następujące kroki:

  1. Kliknij Filtruj.
  2. Kliknij Dodaj z prawej strony opcji „Rozszerzenie” lub „Prefix”.
  3. Wprowadź określone kryterium, na przykład „.exe” w przypadku rozszerzenia lub „~” w przypadku prefiksu.
  4. Kliknij OK, aby zamknąć okno dialogowe konfiguracji, a następnie kliknij ponownie OK, aby zapisać ustawienia.

Włączenie powiadomienia email

Użyj tej funkcji, aby otrzymywać w wiadomościach e-mail szczegóły po ukończeniu zaplanowanej kopii zapasowej.

  1. Kliknij Konfiguracja SMTP.
  2. Wprowadź poprawne informacje o koncie e-mail, i kliknij OK, aby zakończyć.

Włącz wersje plików

Do czego służy funkcja wersji plików?

Włączenie funkcji wersje plików pomoże Ci zachować historię edycji pliku.

Po zakończeniu pierwszej kopii zapasowej wszystkie skopiowane pliki będą przechowywane w miejscu docelowym kopii zapasowej w folderze „latest”. Podczas tworzenia następnej kopii zapasowej Data Replicator 3 automatycznie sprawdzi, czy już istnieje w folderze „latest” jakiś plik o tej samej nazwie. Jeśli plik istnieje, a jego zawartość uległa zmianie, poprzednio zapisany plik w folderze „latest” zostanie przeniesiony do folderu „past” jako starsza wersja, a plik na Twoim komputerze zostanie skopiowany do folderu „latest”, w którym zawsze znajdziesz najnowsze kopie zapasowe plików.

Uwaga:

Plik, o którym mowa powyżej, musi być w tym samym folderze. Jeżeli więc używasz tego samego konta do tworzenia kopii zapasowej folderu, zostanie wygenerowana poprzednia wersja.

Kiedy funkcja wersji plików jest włączona i dokonasz niewłaściwej oraz nieodwracalnej edycji plików lokalnych, możesz znaleźć starsze wersje tych plików w miejscu docelowym kopii zapasowej i łatwo je przywrócić.

Liczba poprzednich wersji

Możesz tu ustawić liczbę poprzednich wersji plików kopii zapasowej, które chcesz przechowywać. Maksimum to 20 wersji. Nazwy plików poprzednich wersji będą miały dodane „v00”, „v01” i tak dalej. Im większy numer, tym nowsza wersja.

Ustaw liczbę poprzednich wersji, aby określić, ile poprzednich wersji będzie przechowywanych w miejscu docelowym kopii zapasowej. Wykonaj następujące kroki:

  1. Zaznacz Włącz wersje plików.
  2. Ustaw liczbę poprzednich wersji i kliknij Zastosuj, aby zapisać ustawienia.

Punkty przywracania

Co to jest punkt przywracania?

Za każdym razem po wykonaniu kopii zapasowej powstanie folder o nazwie odpowiadającej godzinie rozpoczęcia tworzenia kopii zapasowej i zawierający wszystkie pliki tej kopii. Jest to punkt przywracania.

Plik, który nigdy nie był modyfikowany, pojawi się w każdym punkcie przywracania po każdym utworzeniu kopii zapasowej. Na dysku zostanie jednak zajęte miejsce tylko jednego pliku.

Liczba punktów przywracania

Możesz tu ustawić, ile punktów przywracania chcesz przechowywać. Maksymalna liczba to 128 punktów. Kiedy liczba punktów przywracania przekroczy ustawioną wartość, najstarszy punkt przywracania w miejscu docelowym kopii zapasowej zostanie automatycznie usunięty po następnej kopii zapasowej.

Obsługa i ograniczenie
Funkcja punktu przywracania jest obsługiwana tylko w systemach Windows 2000 / Windows XP / Windows 2003 / Windows Vista.

Automatyczne uruchamianie po zalogowaniu

Możesz ustawić automatyczne uruchamianie Synology Data Replicator 3 po zalogowaniu do systemu Windows. Wykonaj następujące kroki:

  1. Zaznacz Auto-start Data Replicator 3 przy uruchomieniu Windows.
  2. Kliknij Zastosuj.

Sprawdź spójność danych przed synchronizacją

Zanim Data Replicator przejdzie do trybu synchronizacji, możesz sprawdzić spójność danych. Wykonaj następujące kroki:

  1. Zaznacz Sprawdź spójność danych przed synchronizacją.
  2. Kliknij Zastosuj.

Uwaga:

Jeżeli nie zaznaczysz tej opcji i zamkniesz Data Replicator, dodane, usunięte lub zmodyfikowane dane mogą nie zostać zsynchronizowane z serwerem po ponownym uruchomieniu programu Data Replicator.

Przechowywanie identycznych plików na serwerze docelowym

Możesz ustawić przechowywanie w folderze „latest” plików identycznych jak na komputerze. Wykonaj następujące kroki:

  1. Zaznacz Kiedy pliki lokalne są usuwane, usuń w tym samym czasie odpowiednie pliki kopii zapasowej.
  2. Kliknij Zastosuj.

Log

Maksymalny rozmiar logu to 512 KB. Kiedy plik logu osiągnie limit, najstarsze 128 KB zostanie usunięte.

  • Zapisywanie logu
    1. Kliknij Zapisz.
    2. Wybierz ścieżkę miejsca docelowego kopii zapasowej, a następnie kliknij Zapisz, aby zakończyć.

  • Czyszczenie logu
    1. Kliknij Wyczyść.
    2. Kliknij Tak w oknie dialogowym potwierdzenia.
