Usando o Data Replicator 3

Clique nas funções abaixo para conhecer o Data Replicator 3.

  • Backup de dados do Computador
  • Restauração de arquivos de backup
  • Configurações avançadas
  • Registro

Backup de dados do Computador

Siga os passos abaixo para configurar seu backup. Você pode também ir a Configurações avançadas para conhecer outras configurações que o ajudarão a personalizar o backup dos dados do computador.

  1. Selecionar um destino de backup

    2. Você pode escolher copiar seus dados do computador para um servidor Synology ou outro lugar. Depois que o backup é efetuado com sucesso, você poderá encontrar os arquivos de backup na pasta atribuída.

    1.1 Backup para um servidor Synology

    1. Clique em Selecionar.
    2. Selecione Servidor Synology e clique em OK.
    3. Na próxima janela, clique no servidor Synology onde a pasta compartilhada de backup está localizada. Então clique em Avançar.
    4. Informe o nome de usuário e senha que pode Ler/Gravar na pasta compartilhada que você escolheu e clique em Avançar.
    5. Clique na pasta compartilhada que deseja utilizar para backup e clique em Concluir.

    1.2 Backup para outro local

    1. Clique em Selecionar.
    2. Selecione Outro Local e clique em Procurar para localizar o destino do backup.
    3. Clique em OK para concluir.

  2. Marque as pastas das quais deseja fazer backup

  3. Escolha um modo de backup

    4. O Data Replicator 3 fornece 3 modos de backup para sua maior conveniência, que são: Imediato, Sinc, e Agendamento.

    3.1 Imediato

    Ao clicar no botão Imediato, todos os arquivos nas pastas selecionadas locais serão imediatamente copiados para o destino do backup.

    3.2 Sinc

    • Sincroniza os dados agora e inicia o monitoramento

      • Clique no botão Sinc e o Data Replicator 3 irá fazer um backup completo de todos os arquivos nas pastas selecionadas para o destino do backup. Depois disso, o Data Replicator 3 irá começar a monitorar estas pastas. À medida que você efetua modificações, o Data Replicator 3 copia apenas os arquivos que tenham mudado.

      Nota:

      1. Depois que o Data Replicator 3 entra em modo de monitoramento, ele permanecerá trabalhando até que você o pare.
      2. O intervalo de tempo entre sincronização de emails refere-se ao tempo entre a conclusão de uma sincronização e o início da próxima. A tarefa pode ser adiada quando o Data Replicator 3 está processando a sincronização de outros arquivos.

    • Parar o monitoramento

      • Para parar o monitoramento, clique no botão Parar Sincronização na janela do programa Data Replicator 3. Se a janela do programa tiver sido minimizada para a bandeja do sistema, dê um duplo clique no ícone do Data Replicator 3 para abri-la.

    3.3 Agendamento

    • Definir agendamento

      • O Data Replicator 3 pode ser ajustado para fazer backups automáticos a uma hora especificada regularmente. Para fazer isso:

      1. Clique em Agendamento.
      2. Ajuste o período de backup e hora de início para o backup agendado, e clique em OK.
      3. Se deseja executar um backup agendado quando não estiver logado, informe o nome de usuário e senha que você está usando ao fazer logon neste computador. Então clique em OK.
        Se não deseja executar o backup agendado quando não estiver logado, clique em Ignorar.

      Nota:

      1. Para efetuar backup enquanto não está logado, a senha de usuário não pode ser vazia.
      2. Para usuários executando o Windows 98 ou Windows ME, o backup agendado pode ser executado enquanto não estiver logado por design.

      Depois que a configuração do agendamento está completa, você pode sair do Synology Data Replicator 3 ou minimizar a janela para a bandeja do sistema. O Synology Data Replicator 3 irá executar o backup agendado automaticamente quando a hora chegar.

    • Excluir agendamento

      • Clique em Excluir Agendamento para parar o backup agendado e remover as configurações de agendamento.

    Quando quiser fazer backup dos arquivos do Outlook, Outlook Express ou Windows Mail, observe:

    • Se está executando o Windows XP, Windows 2003 ou o Windows Vista,
      1. Pelo menos um dos discos de seu computador deve estar em formato NTFS.
      2. Os arquivos do Outlook/Windows Mail devem estar em discos locais.
      3. Você deve fazer logon como administrador do computador.
    • Se está executando outros sistemas que não o Windows XP e Windows 2003, feche o Outlook antes de fazer backup.

      • Suporte & Limitação

      1. Ao pesquisar pelo servidor alvo, o Synology Data Replicator 3 irá utilizar as portas 9999, 9998 ou 9997.
      2. Arquivos compactados e criptografados irão tornar-se arquivos comuns após o backup. As propriedades do arquivo serão apagadas.
      3. Quanto ao tamanho do nome de arquivo, o Synology Data Replicator 3 suporta até 240 bytes.
      4. Se você modificar o nome de seu computador, deve informar a senha de logon de admin de seu computador ao definir o backup agendado para garantir que o Data Replicator 3 funcione corretamente. Se já tiver configurado um backup agendado, remova aquele agendamento e use a senha de admin de seu computador para definir um novo.

Restauração de arquivos de backup

Para restaurar arquivos, você deve primeiro selecionar o destino do backup, e então escolher um dos métodos a seguir para restaurar seus arquivos de backup:

  1. Restaurar procurando

    2. Na maior parte das situações, se você tem certeza de onde estão os arquivos de backup, pode utilizar Restaurar procurando para restaurar seus arquivos:

    1. Clique no botão Iniciar em Restaurar procurando.
    2. Clique em Avançar.
    3. Clique para selecionar a origem da restauração e clique em Avançar.
    4. Escolha listar os arquivos Por versão ou Por ponto de restauração.
    5. Marque as pastas que deseja restaurar e clique em Avançar.
    6. Escolha o local onde os arquivos de backup deverão ser restaurados. Então clique em Concluir.

  2. Restaurar pesquisando

    3. Ser não tem certeza de onde estão os arquivos de backup, ou se apenas quer restaurar arquivos com uma determinada palavra-chave, siga os passos a seguir para restaurar:

    1. Clique no botão Iniciar em Restaurar pesquisando.
    2. Clique em Avançar.
    3. Clique para selecionar a origem da restauração e clique em Avançar.
    4. Digite a palavra-chave que deseja pesquisar no campo de texto "Palavra-chave:".
    5. Escolha a pasta de backup a partir do menu "Backups:".
    6. Clique em Pesquisar.
    7. Selecione os arquivos e pastas que deseja restaurar no quadro direito, e clique em Avançar.
    8. Escolha o local onde os arquivos de backup deverão ser restaurados. Então clique em Concluir.

    Dica sobre restaurar Arquivos do Outlook, Outlook Express e Windows Mail

    Ao escolher um caminho de destino, pode escolher:

    1. Caminho original
      2.

      Ao escolher esta opção, você não precisa importar manualmente os contatos e emails, os dados do computador serão substituídos pelos arquivos restaurados.

    2. Novo caminho
      3.

      Ao escolher esta opção, você pode restaurar os arquivos para um caminho especificado, e importar manualmente os contatos e email com o Outlook/Outlook Express/Windows Mail.

Configurações avançadas

Na seção de Opções, você pode ajustar as configurações avançadas de backup da seguinte forma.

Filtro

Use esta função para efetuar backup preciso de arquivos. A exclusão de arquivos com determinado prefixo ou extensão da lista de backup irá acelerar o backup. Siga os passos para executar:

  1. Clique em Filtro.
  2. Clique em Adicionar no lado direito da opção "Extensão" ou "Prefix".
  3. Informe o critério especificado, por exemplo, ".exe" para extensão ou "~" para prefixo.
  4. Clique em OK para fechar a caixa de diálogo de configuração, e clique em OK novamente para salvar as configurações.

Ativar notificação por e-mail

Use esta função para receber detalhes sobre o backup por email após o backup agendado estar concluído.

  1. Clique em Configuração SMTP.
  2. Informe os dados de e-mail válidos, e clique em OK para concluir.

Habilitar versões de arquivo

Para que serve a função de versão de arquivos?

Habilitar a função de versões de arquivo pode ajudá-lo a manter edições históricas do mesmo arquivo.

Depois que o backup da primeira vez estiver concluído, todos os arquivos copiados serão armazenados na pasta "latest" no destino de backup. A partir do próximo backup, o Data Replicator 3 irá automaticamente verificar se existe algum arquivo com o mesmo nome na pasta "latest". Se existir e o conteúdo do arquivo houver mudado, o arquivo anteriormente copiado na pasta "latest" será movido para a pasta "past" como uma versão mais antiga, e o arquivo em seu computador será copiado para a pasta "latest", onde você pode sempre encontrar os arquivos de backup mais recentes.

Nota:

O arquivo mencionado acima deve estar na mesma pasta. Dessa forma, quando você está utilizando a mesma conta para fazer backup da pasta, a versão anterior será gerada.

Quando a função de versão de arquivo está habilitada e você encontrar edição imprópria e irreversível de seus arquivos locais, poderá encontrar os arquivos de histórico no destino de backup e restaurá-los facilmente.

Número de versões anteriores

Você pode definir aqui quantas versões anteriores de arquivos de backup deseja manter, sendo 20 o número máximo. O nome de arquivo das versões anteriores serão adicionados de "v00", "v01", e assim por diante. Quanto maior o número, mais recente é a versão.

Defina um número de versões anteriores para decidir quantas versões anteriores serão mantidas no destino de backup. Siga os passos para executar:

  1. Marque Habilitar versões de arquivo.
  2. Defina o número de versões anteriores, e clique em Aplicar para salvar as configurações.

Pontos de restauração

O que é um ponto de restauração?

Cada vez que um backup é executado, haverá uma pasta com a hora de início do backup que contém todos os arquivos deste backup. Isso será um ponto de restauração.

O arquivo que nunca tenha sido modificado aparecerá em cada ponto de restauração após cada backup. Entretanto, ocupará o tamanho em disco apenas uma vez.

Número de pontos de restauração

Você pode definir aqui quantos pontos de restauração deseja manter, sendo 128 pontos de restauração o máximo. Quando o número de pontos de restauração exceder o valor definido, os pontos de restauração mais antigos no destino do backup serão automaticamente deletados após o próximo backup.

Suporte & Limitação
A função de ponto de restauração é suportada apenas no Windows 2000 / Windows XP / Windows 2003 / Windows Vista.

Iniciar automaticamente ao efetuar logon

Você pode configurar o Synology Data Replicator 3 para iniciar automaticamente quando você faz logon no Windows. Siga os passos para executar:

  1. Marque Iniciar o Data Replicator 3 ao iniciar o Windows.
  2. Clique em Aplicar.

Verificar a consistência dos dados antes da sincronização

Você pode escolher se quer verificar a consistência dos dados antes que o Data Replicator entre no modo sincronização. Siga os passos para executar:

  1. Marque Verificar a consistência dos dados antes da sincronização.
  2. Clique em Aplicar.

Nota:

Se você desmarcar esta opção e fechar o Data Replicator, os dados adicionados, removidos ou modificados podem não estar sincronizados com o servidor da próxima vez que iniciar o Data Replicator.

Manter arquivos idênticos no servidor destino

Você pode configurar para manter arquivos na pasta "latest" que sejam idênticos aos do seu computador local. Siga os passos para executar:

  1. Marque Quando arquivos locais são removidos, remova os arquivos correspondentes de backup ao mesmo tempo.
  2. Clique em Aplicar.

Registro

O tamanho máximo do arquivo de registro é 512KB; quando o arquivo de registro atingir o limite, os 128KB mais antigos serão removidos.

  • Salvar registro
    1. Clique em Salvar.
    2. Escolha o caminho do destino do backup e clique em Salvar para concluir.

  • Limpar registro
    1. Clique em Limpar.
    2. Clique em Sim na caixa de diálogo de confirmação.
