Bruke Data Replicator 3

Klikk på følgende hovedfunksjoner for å bli kjent med Data Replicator 3.

  • Sikkerhetskopiering av PC-data
  • Gjenopprette sikkerhetskopifiler
  • Avanserte innstillinger
  • Logg

Sikkerhetskopiering av PC-data

Vennligst følg trinnene nedenfor for å sette opp sikkerhetskopieringen. Du kan også gå til Avanserte innstillinger for å bli kjent med andre innstillinger, som vil hjelpe deg å tilpasse sikkerhetskopieringen av PC-data til dine personlige behov.

  1. Velg et sikkerhetskopimål

    2. Du kan velge å sikkerhetskopiere PC-dataene dine til en Synology server eller en annen adresse. Etter at sikkerhetskopieringen er vellykket gjennomført, vil du finne de sikkerhetskopierte filene i den tildelte mappen.

    1.1 Sikkerhetskopier til en Synology server

    1. Klikk Velg.
    2. Velg Synology Server og klikk OK.
    3. I neste vindu klikker du på den Synology serveren hvor den delte mappen for sikkerhetskopier befinner seg. Deretter klikker du Neste.
    4. Skriv inn brukernavn og passordet som kan Lese fra/Skrive til den delte mappen du nettopp har valgt, og klikk Neste.
    5. Klikk på den delte mappen du ønsker å bruke til sikkerhetskopier og klikk deretter Fullfør.

    1.2 Sikkerhetskopiering til en annen adresse

    1. Klikk Velg.
    2. Velg Annen adresse, og klikk Bla gjennom for å finne sikkerhetskopimålet.
    3. Klikk OK for å fullføre.

  2. Hak av mappene du ønsker å sikkerhetskopiere

  3. Velg sikkerhetskopieringsmodus

    4. Data Replicator 3 lar deg for enkelhets skyld velge mellom 3 sikkerhetskopieringsmodi: Umiddelbar, Synkronisert og Tidsbestemt.

    3.1 Umiddelbar

         Ved å klikke Umiddelbar-knappen, vil alle filene i den/de valgte lokale mappen(e) umiddelbart bli sikkerhetskopiert til det valgte sikkerhetskopimålet.

    3.2 Synkronisert

    • Synkroniser data nå og begynn overvåkning

      • Klikk Synkroniser-knappen, og Data Replicator 3 vil foreta en komplett sikkerhetskopiering av alle filene i den/de valgte mappen(e) til sikkerhetskopimålet. Deretter vil Data Replicator 3 begynne å overvåke mappen(e). Når du foretar endringer, sikkerhetskopierer Data Replicator 3 bare filer som er endret.

      Merk:

      1. Etter at Data Replicator 3 er gått i overvåkningsmodus, vil den fortsette å arbeide inntil du stopper.
      2. Tidsintervallet mellom e-postsynkronisering er tiden fra synkroniseringen er fullført til neste synkronisering starter. Oppgaven kan bli utsatt hvis Data Replicator 3 gjennomfører synkronisering av andre filer.

    • Stopp overvåkning

      • For å stoppe overvåkningen, klikk Stopp synkronisering-knappen i programvinduet til Data Replicator 3. Dersom programvinduet har blitt minimalisert og lagt i systemfeltet, vennligst dobbelklikk på ikonet til Data Replicator 3 for å åpne programvinduet.

    3.3 Tidsplan

    • Stille inn tidstabell

      • Data Replicator 3 kan konfigureres til å foreta automatiske sikkerhetskopieringer jevnlig på oppgitte tidspunkt. Slik gjør du dette:

      1. Klikk Tidstabell.
      2. Still inn sikkerhetskopieringsperioden og starttid for Tidsinnstilt sikkerhetskopiering og klikk OK.
      3. Dersom du ønsker å utføre Tidsinnstilt sikkerhetskopiering når du er logget av, vennligst skriv inn brukernavn og passordet du bruker når du logger på denne datamaskinen. Deretter klikk OK.
        Dersom du ikke ønsker å utføre Tidsinnstilt sikkerhetskopiering når du er logget av, vennligst klikk Hopp over.

      Merk:

      1. For å foreta sikkerhetskopiering når du er logget av, kan brukernavn og passord ikke stå tomt.
      2. For brukere som kjører Windows 98 eller Windows ME, er muligheten til utføring av Tidsinnstilt sikkerhetskopiering når du er logget av innebygget i programmet.

      Når tidsinnstillingen er fullført, kan du avslutte Synology Data Replicator 3 eller minimalisere programvinduet til systemstatusfeltet. Synology Data Replicator 3 vil automatisk utføre tidsbestemt sikkerhetskopiering når tiden er inne.

    • Slette tidstabell

      • Klikk Slett tidstabell for å stoppe Tidsinnstilt sikkerhetskopiering og fjerne tidsinnstillingene.

    Når du ønsker å sikkerhetskopiere Outlook-, Outlook Express- eller Windows Mail-filer, vennligst merk:

    • Hvis du kjører Windows XP, Windows 2003 eller Windows Vista,
      1. Minst en av stasjonene på PC-en må være i NTFS-formatet.
      2. Outlook-/Windows Mail-filer må være på lokale stasjoner.
      3. Du må logge inn som PC-ens administrator.
    • Hvis du kjører andre operativsystemer enn Windows XP og Windows 2003, må du lukke Outlook før du sikkerhetskopierer Outlook.

      • Støtte & begrensninger

      1. Når du søker etter målserver, vil Synology Data Replicator 3 bruke port 9999 eller 9998 eller 9997.
      2. Komprimerte filer og krypterte filer vil bli vanlige filer etter sikkerhetskopiering. Filegenskapene vil bli slettet.
      3. Når det gjelder lengden på filnavn, støtter Synology Data Replicator 3 opptil 240 byte.
      4. Hvis du endrer navnet på PC-en, må du angi administratorpassordet for PC-en når du setter opp Tidsbestemt sikkerhetskopiering, for å være sikker på at Data Replicator 3 fungerer på riktig måte. Hvis du allerede har satt opp en tidstabell for sikkerhetskopiering, må du slette den tidstabellen og bruke administratorpassordet for PC-en til å sette opp en ny tidstabell.

Gjenopprette sikkerhetskopifiler

For å gjenopprette sikkerhetskopifiler må du velge sikkerhetskopimål først, og deretter en av de følgende metodene for å gjenopprette de sikkerhetskopierte filene dine:

  1. Gjenopprett ved å bla gjennom

    2. Dersom du er usikker på hvor de sikkerhetskopierte filene befinner seg, kan du i de fleste tilfeller bruke Gjenopprett ved å bla gjennom for å gjenopprette filene dine:

    1. Klikk Start-knappen under Gjenopprett ved å bla gjennom.
    2. Klikk på Neste.
    3. Klikk for å velge gjenopprettingskilde, og klikk deretter Neste.
    4. Klikk for å liste opp filer Etter versjon eller Etter gjenopprettingspunkt.
    5. Hak av mappen(e) som du ønsker å gjenopprette, og klikk Neste.
    6. Velg adressen som de sikkerhetskopierte filene skal gjenopprettes til. Deretter klikker du Fullfør.

  2. Gjenopprett ved søk

    3. Dersom du ikke er sikker på hvor de sikkerhetskopierte filene befinner seg, eller du ønsker kun å gjenopprette filer som inneholder bestemte nøkkelord, vennligst gjennomfør følgende trinn for å gjenopprette:

    1. Klikk Start-knappen under Gjenopprett ved søk.
    2. Klikk på Neste.
    3. Klikk for å velge gjenopprettingskilde, og klikk deretter Neste.
    4. Skriv inn nøkkelordet du vil søke etter i ”Nøkkelord:” tekstfeltet.
    5. Velg sikkerhetskopieringsmappe fra ”Sikkerhetskopier:” nedtrekksmenyen.
    6. Klikk Søk.
    7. Velg filene og mappene du ønsker å gjenopprette i den rammen til høyre, og klikk Neste.
    8. Velg adressen som de sikkerhetskopierte filene skal gjenopprettes til. Deretter klikker du Fullfør.

    Tips for gjenoppretting av Outlook-, Outlook Express- og Windows Mail- filer

         Når du velger en målsti, kan du velge:

    1. Original sti
      2.

      Hvis du velger dette alternativet trenger du ikke importere e-post og adressebok manuelt, da PC-data vil bli erstattet av de gjenopprettede filene.

    2. Ny sti
      3.

      Hvis du velger dette alternativet, kan du gjenopprette dataene til en bestemt sti og importere e-post og adressebok manuelt med Outlook/Outlook Express/Windows Mail.

Avanserte innstillinger

I seksjonen for Alternativer, kan du stille inn avanserte sikkerhetskopiinnstillinger, som følger:

Filter

Bruk denne funksjonen for å foreta presise sikkerhetskopier. Ekskludering av filer med bestemte forstavelser eller etternavn fra sikkerhetskopilisten, vil hjelpe deg å gjennomføre sikkerhetskopieringen hurtigere. Følg trinnene for å fortsette:

  1. Klikk Filter.
  2. Klikk Legg til på høyre side for alternativene for ”Filetternavn” eller ”Prefix”.
  3. Oppgi kriterier, for eksempel ”.exe” for filetternavn eller ”~” for forstavelse.
  4. Klikk OK for å lukke dialogvinduet for innstillinger, og klikk OK igjen for å lagre innstillingene.

Aktiver e-postvarsling

Bruk denne funksjonen for å motta sikkerhetskopieringsdetaljer på e-post etter Tidsinnstilt sikkerhetskopiering er ferdig.

  1. Klikk SMTP-oppsett.
  2. Skriv inn gyldig e-postinformasjon og klikk OK for å fullføre.

Aktiver filversjoner

Hva er filversjonsfunksjonen til?

Aktivering av filversjonsfunksjonen kan hjelpe deg å ta vare på redigeringshistorien til den samme filen.

Når den første sikkerhetskopieringen er fullført, vil alle sikkerhetskopierte filer bli lagret i den ”latest” mappen i sikkerhetskopimålet. Fra og med den neste sikkerhetskopieringen, vil Data Replicator 3 automatisk sjekke om noen filer med det samme navnet finnes i den ”latest” mappen. Hvis det gjør det og innholdet i filene er endret, vil den sikkerhetskopierte filen i den latest mappen bli flyttet til en past mappe som en eldre versjon, mens filen på datamaskinen blir sikkerhetskopiert til den latest mappen, der du alltid finner de nyeste sikkerhetskopierte filene.

Merk:

Filen det refereres til ovenfor må være i den samme mappen. Slik at når du bruker den samme kontoen til å sikkerhetskopiere mappen, vil en tidligere versjon bli generert.

Når filversjonsfunksjonen er aktivert og du opplever uriktig og uforanderlig redigering av de lokale filene dine, kan du finne filhistorien i sikkerhetskopimålet og enkelt gjenopprette filene.

Antall tidligere versjoner

Her kan du stille inn hvor mange tidligere versjoner av sikkerhetskopierte filer du ønsker å beholde. 20 versjoner er maksimalt antall. Filnavnet til tidligere versjoner av filene vil få lagt til ”v00”, ”v01”, osv. Jo høyere versjonsnummeret er, desto nyere er den.

Still inn antall tidligere versjoner for å bestemme hvor mange tidligere versjoner som skal beholdes i sikkerhetskopimålet. Følg trinnene for å fortsette:

  1. Hak av Aktiver filversjoner.
  2. Skriv inn et antall tidligere versjoner og klikk Bruk for å lagre innstillingene.

Gjenopprettingspunkter

Hva er gjenopprettingspunkter?

Hver gang en sikkerhetskopiering utføres, vil det bli opprettet en mappe navngitt som sikkerhetskopieringens starttid, som inneholder alle filene i denne sikkerhetskopieringen. Dette vil bli et gjenopprettingspunkt.

Filen som aldri har vært endret, vil vises ved hvert gjenopprettingspunkt etter hver sikkerhetskopiering. Imidlertid vil kun en fildiskstørrelse faktisk bli brukt.

Antall gjenopprettingspunkter

Her kan du stille inn hvor mange gjenopprettingspunkter du ønsker å beholde. 128 gjenopprettingspunkter er maksimalt antall. Når antallet gjenopprettingspunkt overskrider den innstilte verdien, vil det eldste gjenopprettingspunktet automatisk bli slettet etter neste sikkerhetskopiering.

Støtte & begrensninger
Funksjonen for gjenopprettingspunkt støttes bare for Windows 2000 / Windows XP / Windows 2003 / Windows Vista.

Autostart ved pålogging

Du kan stille inn Synology Data Replicator 3 til å starte automatisk når du logger deg på Windows. Følg trinnene for å fortsette:

  1. Hak av Autostart Data Replicator 3 når Windows starter.
  2. Klikk Bruk.

Sjekk dataoverensstemmelse før synk

Du kan bestemme om dataoverensstemmelse skal sjekkes før Data Replicator går i synk-modus. Følg trinnene for å fortsette:

  1. Merk av Sjekk dataoverensstemmelse før synkronisering starter.
  2. Klikk Bruk.

Merk:

Dersom du ikke velger dette alternativet og lukker Data Replicator, vil data som er lagt til, fjernet eller endret ikke bli synkronisert til serveren når du starter Data Replicator neste gang.

Beholde identiske filer på målserver

Du kan stille inn programmet slik at filer i den "latest" mappen skal være identiske med dem på den lokale datamaskinen. Følg trinnene for å fortsette:

  1. Hak av Når lokale filer slettes, slett de motsatte sikkerhetskopifilene samtidig.
  2. Klikk Bruk.

Logg

Maksimalstørrelse for loggfilen er 512 KB. Når loggfilen når denne grensen, slettes de eldste 128 KB.

  • Lagre loggen
    1. Klikk Lagre.
    2. Velg målsti for sikkerhetskopimål, og klikk Lagre for å fullføre.

  • Fjern loggen
    1. Klikk på Fjern.
    2. Klikk Ja i dialogboksen for bekreftelse.
