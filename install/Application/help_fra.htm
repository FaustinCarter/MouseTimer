Utilisation de Data Replicator 3

Cliquez sur les principales fonctions suivantes afin d’apprendre à connaître Data Replicator 3.

  • Sauvegarde des données d’un PC
  • Restauration des fichiers de sauvegarde
  • Paramètres avancés

Sauvegarde des données d’un PC

Veuillez utiliser les étapes suivantes pour configurer votre sauvegarde. Vous pouvez également consulter Paramètres avancés pour en apprendre davantage sur les paramètres supplémentaires qui vous aideront à personnaliser la sauvegarde des données de votre PC.

  1. Sélectionnez une destination de sauvegarde

    2. Vous pouvez choisir de sauvegarder les données de votre PC sur un serveur Synology ou dans un autre emplacement. Après avoir terminé avec succès la sauvegarde, vous serez en mesure de trouver les fichiers de sauvegarde dans le dossier assigné.

    1.1 Sauvegarde sur un serveur Synology

    1. Cliquez sur Sélectionner.
    2. Sélectionnez Serveur Synology, et cliquez sur OK.
    3. Dans la fenêtre suivante, cliquez sur le serveur Synology où le dossier partagé de la sauvegarde se trouve. Puis cliquez sur Suivant.
    4. Entrez le nom d’utilisateur et le mot de passe ayant un accès en lecture/écriture au dossier partagé que vous venez de sélectionner et cliquez sur Suivant.
    5. Cliquez sur le dossier partagé que vous souhaitez utiliser pour la sauvegarde et cliquez sur Terminer.

    1.2 Sauvegarde dans un autre emplacement

    1. Cliquez sur Sélectionner.
    2. Sélectionnez Autre emplacement et cliquez sur Parcourir afin de localiser l'emplacement de sauvegarde.
    3. Cliquez sur OK pour terminer.

  2. Sélectionner les dossiers à sauvegarder

  3. Choisir un mode de sauvegarde

    4. Data Replicator 3 fournit les 3 modes de sauvegarde suivants pour plus de commodité : Immédiat, Sync et Planifié.

    3.1 Immédiat

    Si vous cliquez sur le bouton Immédiat, tous les fichiers sélectionnés dans le ou les dossiers locaux seront immédiatement sauvegardés dans l'emplacement de sauvegarde.

    3.2 Sync

    • Synchroniser les données maintenant et commencer la surveillance

      • Cliquez sur le bouton Sync et Data Replicator 3 crée une sauvegarde complète de tous les fichiers du dossier sélectionné dans l'emplacement de sauvegarde. Data Replicator 3 commencera ensuite la surveillance de ces dossiers. Lorsque vous modifiez des fichiers, Data Replicator 3 les sauvegarde.

      Remarque :

      1. Une fois que Data Replicator 3 est entré en mode de surveillance, il continuera à fonctionner jusqu’à ce que vous l’arrêtiez.
      2. L’intervalle entre deux synchronisation du courrier est le temps entre la fin d’une opération et le début de la suivante. Cette tâche peut être reportée lorsque Data Replicator 3 synchronise d’autre fichiers.

    • Arrêter la surveillance

      • Pour arrêter la surveillance, cliquez sur le bouton Arrêter synchronisation sur la fenêtre de Data Replicator 3. Si vous avez réduit la fenêtre du programme dans la barre d'état système, veuillez double-cliquer sur l'icône de Data Replicator 3 pour l'ouvrir.

    3.3 Programmée

    • Définir la programmation

      • Data Replicator 3 peut créer des sauvegardes automatiques à des intervalles réguliers. Pour ce faire :

      1. Cliquez sur Planification.
      2. Sélectionnez le moment et l'heure de début de la sauvegarde pour la sauvegarde programmée. Cliquez sur OK.
      3. Si vous voulez effectuer une sauvegarde programmée en étant déconnecté, veuillez entrer le nom d’utilisateur et le mot de passe que vous utilisez pour la connexion à cet ordinateur. Cliquez ensuite sur OK.
        Si vous ne désirez pas effectuer de sauvegarde programmée en étant déconnecté, veuillez cliquer sur Ignorer.

      Remarque :

      1. Pour effectuer la sauvegarde en état de déconnexion, le mot de passe utilisateur ne peut pas être vide.
      2. Pour les utilisateurs sous Windows 98 ou Windows Me, une sauvegarde programmée peut être effectuée en état de déconnexion en raison de la conception du système d’exploitation.

      Lorsque la configuration du programme est terminée, vous pouvez quitter Synology Data Replicator 3 ou réduire la fenêtre du programme dans la zone de notification. Synology Data Replicator 3 effectuera la sauvegarde programmée lorsque le temps se sera écoulé.

    • Supp. planification

      • Cliquez sur Supp. planification pour arrêter la sauvegarde programmée et pour supprimer les paramètres du programme.

    Lorsque vous souhaitez sauvegarder Outlook, Outlook Express ou les fichiers Windows Mail, veuillez prêter attention aux remarques suivantes :

    • Si votre système utilise Windows XP, Windows 2003 ou Windows Vista,
      1. Au moins un des disques de votre ordinateur doit être au format NTFS.
      2. Les fichiers Outlook/Windows Mail doivent se trouver sur les disques locaux.
      3. L'utilisateur doit se connecter en tant qu'administrateur de l'ordinateur.
    • Si vous utilisez des systèmes d'exploitation autre que Windows XP et Windows 2003, veuillez fermer Outlook avant d'effectuer sa sauvegarde.

      • Prise en charge et limitations

      1. Lors de la recherche du serveur cible, Synology Data Replicator 3 utilisera le port 9999, 9998 ou 9997.
      2. Les fichiers comprimés et les fichiers cryptés deviendront des fichiers normaux après la sauvegarde. La propriété de fichier sera effacée.
      3. Pour la longueur du nom de fichier, Synology Data Replicator 3 prend en charge jusqu'à 240 octets.
      4. Si vous changez le nom de votre ordinateur, vous devez entrer le mot de passe administrateur de votre PC lors de la configuration d'une sauvegarde programmée afin d'assurer que Data Replicator 3 fonctionne correctement. Si vous avez déjà configuré un programme de sauvegarde, supprimez-le et utilisez le mot de passe administrateur de votre PC pour en définir un nouveau.

Restauration des fichiers de sauvegarde

Pour restaurer des fichiers de sauvegarde, vous devez d’abord sélectionner un emplacement de sauvegarde et ensuite choisir une des méthodes suivantes afin de restaurer vos fichiers de sauvegarde :

  1. Restaurer par navigation

    2. Dans la plupart des situations, si vous êtes certain de l’emplacement des fichiers, vous pouvez utiliser l’option Restaurer par navigation pour restaurer vos fichiers :

    1. Cliquez sur le bouton Démarrer sous Restaurer par navigation.
    2. Cliquez sur Suivant.
    3. Cliquez pour sélectionner la source de restauration et cliquez sur Suivant.
    4. Vous pouvez choisir de lister les fichiers Par version ou Par point de restauration.
    5. Sélectionnez le ou les dossiers que vous voulez restaurer et cliquez sur Suivant.
    6. Choisissez l’emplacement où les fichiers de sauvegarde seront restaurés. Cliquez ensuite sur Terminer.

  2. Restaurer par recherche

    3. Si vous n’êtes pas sûr de l’endroit où se trouvent les fichiers de sauvegarde, ou si vous voulez uniquement restaurer les fichiers correspondants à un certain mot clé, veuillez suivre les étapes suivantes :

    1. Cliquez sur le bouton Démarrer sous Restaurer par recherche.
    2. Cliquez sur Suivant.
    3. Cliquez pour sélectionner la source de restauration et cliquez sur Suivant.
    4. Saisissez le mot-clé à rechercher dans le champ « Mot-clé » : champ texte.
    5. Choisissez le dossier de sauvegarde dans le menu déroulant « Sauvegardes » : menu déroulant.
    6. Cliquez sur Recherche.
    7. Sélectionnez les fichiers et dossiers que vous voulez restaurer dans le cadre de droite et cliquez sur Suivant.
    8. Choisissez l’emplacement où les fichiers de sauvegarde seront restaurés. Cliquez ensuite sur Terminer.

    Conseils pour la restauration de fichiers Outlook, Outlook Express et Windows Mail

    Lorsque vous choisissez un chemin cible, vous pouvez choisir parmi les options suivantes :

    1. Chemin d’origine
      2.

      Lorsque vous choisissez cette option, vous ne devez pas importer les messages et le carnet d'adresses manuellement, ces données sont remplacées par les fichiers restaurés.

    2. Nouveau chemin
      3.

      Lorsque vous choisissez cette option, vous pouvez restaurer les données dans un emplacement précisé et importer les messages et le carnet d'adresses manuellement dans Outlook/Outlook Express/Windows Mail.

Paramètres avancés

Dans la section Options, il vous est possible de configurer les paramètres de sauvegarde avancés suivants :

Filtrer

Utilisez cette fonction pour sauvegarder des fichiers de manière précise. L’exclusion de fichiers comprenant certains préfixes ou extension de la liste de sauvegarde peut vous aider à accélérer la sauvegarde. Procédez comme suit :

  1. Cliquez sur Filtre.
  2. Cliquez sur Ajouter à droite de l'option « Extension » ou « Prefix ».
  3. Saisissez un critère, par exemple « .exe » pour une extension ou « ~ » comme préfixe.
  4. Cliquez sur OK pour fermer la boîte de dialogue de configuration et cliquez sur OK à nouveau pour enregistrer les paramètres.

Activer la notification par courriel

Utilisez cette fonction pour recevoir les informations détaillées au sujet d’une sauvegarde programmée après qu’elle ait été réalisée.

  1. Cliquez sur Configuration SMTP.
  2. Entrez des informations de courrier électronique valides et cliquez sur OK pour terminer.

Activer versions de fichier

À quoi sert la fonction versions de fichier ?

Lorsque vous activez la fonction versions de fichier, cela peut vous aider à garder l’historique des éditions d’un même fichier.

Après avoir terminé la première sauvegarde, tous les fichiers de sauvegarde seront stockés dans le dossier « latest » de l'emplacement de sauvegarde. A partir de la seconde sauvegarde, Data Replicator 3 vérifiera automatiquement si un fichier avec le même nom existe déjà dans le dossier « latest ». S'il existe et si son contenu a changé, le fichier se trouvant dans le dossier « latest » est déplacé dans le dossier « past » et le fichier sur votre ordinateur est sauvegardé dans le dossier « latest », qui contient toujours la version la plus récente de tous les fichiers.

Remarque :

Le fichier auquel on se rapporte ci-dessus doit être dans le même dossier. Par conséquent, lorsque vous utilisez le même compte pour sauvegarder un dossier, une ancienne version sera générée.

Si la fonction versions de fichier est activée, lorsque des fichiers locaux ont été modifiés de manière irréversible et inadéquate, vous pouvez en retrouver l'historique dans l’emplacement de sauvegarde et les restaurer sans problèmes.

Nombre de versions précédentes

Cette option vous permet de définir le nombre de versions précédentes de fichiers de sauvegarde vous souhaitez garder, 20 versions correspond au maximum. L'inscription v00, v01, etc. sera ajouté aux noms de fichier correspondant à une version précédente. Plus le numéro de version est élevé, plus le fichier est récent.

Définissez un nombre de versions précédentes pour déterminer le nombre de copies conservées dans l'emplacement de sauvegarde. Procédez comme suit :

  1. Cochez Activer versions de fichier.
  2. Définissez un nombre de versions précédentes et cliquez sur Appliquer pour enregistrer les paramètres.

Points de restauration

Qu’est-ce qu’un point de restauration ?

Chaque fois qu’une sauvegarde est effectuée, un dossier sera nommé selon l’heure de départ de la sauvegarde. Ce dossier contient tous les fichiers de cette sauvegarde. Il s’agit d’un point de restauration.

Le fichier qui n’a jamais été modifié s'affichera dans chaque point de restauration après chaque sauvegarde. Cependant, il occupera l'espace d'un seul fichier.

Nombre de points de restauration

Cette option vous permet de définir le nombre de points de restauration à conserver, 128 constitue le maximum. Lorsque le nombre de points de restauration excède la valeur définie, les copies les plus anciennes seront automatiquement supprimées après chaque sauvegarde.

Prise en charge et limitations
La fonction de point de restauration n'est prise en charge que par Windows 2000 / Windows XP / Windows 2003 / Windows Vista.

Démarrage automatique lors de l'identification

Vous pouvez démarrer Synology Data Replicator 3 automatiquement au démarrage de Windows. Procédez comme suit :

  1. Cochez Exécuter Data Replicator 3 automatiquement à l’ouverture de ma session Windows.
  2. Cliquez sur Appliquer.

Contrôlez la parité des données avant la synchronisation

Vous pouvez décider de contrôler la parité des données avant que Data Replicator entre en mode synchronisé. Procédez comme suit :

  1. Cochez Contrôler la parité des données avant de commencer la synchronisation.
  2. Cliquez sur Appliquer.

Remarque :

Si vous décochez cette option et que vous fermez Data Replicator, les données ajoutées, supprimées, ou modifiées peuvent ne pas être synchronisées avec le serveur la prochaine fois que vous lancerez Data Replicator.

Garder les fichiers identiques sur le serveur de destination

Vous pouvez choisir de conserver les fichiers dans le dossier « latest » pour qu'ils soient identiques à ceux sur votre ordinateur local. Procédez comme suit :

  1. Cochez Lorsque des fichiers locaux sont éliminés, supprimer également au même moment les fichiers de sauvegarde correspondants.
  2. Cliquez sur Appliquer.

Journal

La taille maximale du journal est de 512 Ko, lorsque le journal atteint cette limite, les premiers 128 Ko sont supprimés.

  • Enregistrer le journal
    1. Cliquez sur Sauvegarder.
    2. Choisissez l'emplacement de la destination de sauvegarde et cliquez sur Enregistrer pour terminer.

  • Effacer le journal
    1. Cliquez sur Effacer.
    2. Cliquez sur Oui sur la boîte de dialogue de confirmation.
