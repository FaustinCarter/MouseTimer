Uso de Data Replicator 3

Haga clic en las siguientes funciones principales para conocer Data Replicator 3.

  • Copia de seguridad de datos en PC
  • Restauración de archivos de copia de seguridad
  • Configuración avanzada
  • Registro

Copia de seguridad de datos en PC

Siga estos pasos para configurar la copia de seguridad. También puede ir a Configuración avanzada para conocer otros ajustes que le ayudarán a personalizar su copia de seguridad de datos del PC.

  1. Seleccione un destino para la copia de seguridad

    2. Puede hacer una copia de seguridad de los datos del PC en un servidor Synology o en otra ubicación. Una vez realizada correctamente la copia de seguridad, podrá buscar los archivos de copia de seguridad en la carpeta que tenga asignada.

    1.1 Copia de seguridad en un servidor Synology

    1. Haga clic en Seleccionar.
    2. Seleccione Servidor Synology y haga clic en OK.
    3. En la siguiente ventana, haga clic en el servidor Synology en el que se encuentra la carpeta compartida para la copia de seguridad. A continuación, haga clic en Siguiente.
    4. Introduzca el nombre de usuario y contraseña que puede Leer/Escribir en la carpeta compartida que acaba de elegir y haga clic en Siguiente.
    5. Haga clic en la carpeta compartida que desee usar para la copia de seguridad y haga clic en Terminar.

    1.2 Copia de seguridad en otra ubicación

    1. Haga clic en Seleccionar.
    2. Seleccione Otra ubicación y haga clic en Examinar para localizar el destino de copia de seguridad.
    3. Haga clic en OK para terminar.

  2. Marque las carpetas de las que desea realizar una copia de seguridad

  3. Seleccione un modo de copia de seguridad

    4. Data Replicator 3 proporciona los 3 modos siguientes de copia de seguridad: Inmediata, Sincronizar y Programa.

    3.1 Inmediata

         Al hacer clic en el botón Inmediata, se hará inmediatamente una copia de seguridad en el destino de copia de seguridad de todos los archivos que se encuentren en las carpetas locales seleccionadas.

    3.2 Sincronizar

    • Sincronizar los datos en este momento y comenzar la supervisión

      • Haga clic en el botón Sincronizar; Data Replicator 3 realizará una completa copia de seguridad en el destino de copia de seguridad de todos los archivos que se encuentren en las carpetas seleccionadas. A continuación, Data Replicator 3 comenzará a supervisar dichas carpetas. Según va realizando cambios, Data Replicator 3 hace una copia de seguridad únicamente de los archivos que han cambiado.

      Observación:

      1. Una vez que Data Replicator 3 entra en modo de supervisión, seguirá funcionando hasta que lo detenga.
      2. El intervalo de tiempo entre la sincronización del correo es el tiempo desde que termina una sincronización hasta que comienza la siguiente. La tarea se puede posponer si Data Replicator 3 está procesando la sincronización de otros archivos.

    • Detener la supervisión

      • Para detener la supervisión, haga clic en el botón Detener sincronización en la ventana del programa Data Replicator 3. Si la ventana del programa se ha minimizado en la bandeja del sistema, haga doble clic en el icono de Data Replicator 3 para abrirla.

    3.3 Programa

    • Establecer el programa

      • Data Replicator 3 puede configurarse para que realice copias de seguridad automáticas de manera periódica a la hora especificada. Para ello:

      1. Haga clic en Programa.
      2. Defina el período de copia de seguridad y la hora final para Copia de seguridad programada y haga clic en OK.
      3. Si desea realizar Copia de seguridad programada durante el tiempo de desconexión, introduzca el nombre de usuario y la contraseña que utiliza para iniciar sesión en este equipo. A continuación, haga clic en OK.
        Si no desea realizar la Copia de seguridad programada durante el tiempo de desconexión, haga clic en Omitir.

      Observación:

      1. Para realizar la copia de seguridad durante el tiempo de desconexión, la contraseña de usuario no puede estar vacía.
      2. Para usuarios de Windows 98 o Windows ME, Copia de seguridad programada se puede ejecutar durante el tiempo de desconexión.

      Cuando finalice la configuración, podrá salir de Synology Data Replicator 3 o minimizar la ventana del programa en la bandeja del sistema. Synology Data Replicator 3 realizará automáticamente la Copia de seguridad programada cuando llegue el momento.

    • Borrar programa

      • Haga clic en Borrar programa para detener Copia de seguridad programada y copiar la configuración del programa.

    Si desea realizar una copia de seguridad de archivos de Outlook, Outlook Express o Windows Mail, tenga en cuenta lo siguiente:

    • Si ejecuta Windows XP, Windows 2003 o Windows Vista,
      1. Al menos una de las unidades del PC deben estar en formato NTFS.
      2. Los archivos de Outlook/Windows Mail deberán estar en unidades locales.
      3. Debe iniciar sesión como administrador del PC.
    • Si ejecuta sistemas operativos distintos de Windows XP y Windows 2003, cierre Outlook antes de hacer una copia de seguridad de Outlook.

      • Soporte y Limitaciones

      1. Al buscar el servidor de destino, Synology Data Replicator 3 utilizará el puerto 9999, 9998 o el 9997.
      2. Los archivos comprimidos y cifrados se convertirán en archivos normales después de la copia de seguridad. La propiedad de los archivos se borrará.
      3. Para la longitud del nombre de archivo, Synology Data Replicator 3 admite hasta 240 bytes.
      4. Si cambia el nombre del PC, deberá introducir la contraseña de inicio de sesión del administrador del PC al configurar Copia de seguridad programada para garantizar que Data Replicator 3 se ejecute de manera correcta. Si ya ha configurado un programa de copia de seguridad, bórrelo y utilice la contraseña de inicio de sesión del administrador de PC para configurar una nueva.

Restauración de archivos de copia de seguridad

Para restaurar archivos de copia de seguridad, deberá seleccionar primero un destino para la copia de seguridad y elegir uno de los siguientes métodos para restaurar los archivos de copia de seguridad:

  1. Restaurar explorando

    2. En la mayoría de las situaciones, si está seguro de dónde se encuentran los archivos de copia de seguridad, puede utilizar Restaurar explorando para restaurar sus archivos:

    1. Haga clic en el botón Iniciar situado debajo de Restaurar explorando.
    2. Haga clic en Siguiente.
    3. Haga clic para seleccionar el origen de restauración y, a continuación, haga clic en Siguiente.
    4. Seleccione mostrar los archivos Por versión o Por punto de restauración.
    5. Marque las carpetas que desee restaurar y haga clic en Siguiente.
    6. Seleccione la ubicación a la que se van a restaurar los archivos de copia de seguridad. A continuación, haga clic en Terminar.

  2. Restaurar buscando

    3. Si no está seguro de dónde se encuentran los archivos de copia de seguridad, o si únicamente desea restaurar los archivos que tengan una determinada palabra clave, siga estos pasos para restaurar:

    1. Haga clic en el botón Iniciar situado debajo de Restaurar buscando.
    2. Haga clic en Siguiente.
    3. Haga clic para seleccionar el origen de restauración y, a continuación, haga clic en Siguiente.
    4. Escriba la palabra clave que desee buscar en el campo de texto "Palabra clave:".
    5. Elija la carpeta de copia de seguridad en el menú desplegable "Copias de seguridad:".
    6. Haga clic en Buscar.
    7. Seleccione los archivos y carpetas que desea restaurar en el cuadro de la derecha y haga clic en Siguiente.
    8. Seleccione la ubicación a la que se van a restaurar los archivos de copia de seguridad. A continuación, haga clic en Terminar.

    Sugerencias sobre la restauración de archivos de Outlook, Outlook Express y Windows Mail

         A la hora de elegir una ruta de destino, puede elegir:

    1. Ruta original
      2.

      Si elige esta opción, no será necesario que importe manualmente el correo y la libreta de direcciones, pues los datos del PC serán reemplazados por los archivos restaurados.

    2. Ruta nueva
      3.

      Si elige esta opción, puede restaurar los datos a una ruta especificada e importar manualmente el correo y la libreta de direcciones con Outlook, Outlook Express y Windows Mail.

Configuración avanzada

En la sección Opciones, puede configurar la configuración de copia de seguridad avanzada como sigue.

Filtrar

Utilice esta función para realizar con precisión la copia de seguridad de los archivos. Si excluye de la lista de copia de seguridad los archivos que contengan un determinado prefijo o extensión, la copia de seguridad se realizará antes. Siga estos pasos para continuar:

  1. Haga clic en Filtrar.
  2. Haga clic en Agregar a la derecha de la opción "Extensión" o "Prefix".
  3. Escriba el criterio especificado; por ejemplo, ".exe" para extensiones o "~" para prefijo.
  4. Haga clic en OK para cerrar el cuadro de diálogo de configuración y de nuevo en OK para guardar la configuración.

Habilitar notificación de correo electrónico

Utilice esta función para recibir detalles de la copia de seguridad por correo electrónico una vez finalizada Copia de seguridad programada.

  1. Haga clic en Configuración de SMTP.
  2. Introduzca información válida de correo electrónico y haga clic en OK para finalizar.

Habilitar versiones de archivos

¿Para qué sirve la función de versiones de archivos?

La función Habilitar versiones de archivos puede ayudarle a conservar el historial de ediciones del mismo archivo.

Una vez finalizada la primera copia de seguridad, todos los archivos de copia de seguridad se almacenarán en la carpeta “latest” en el destino de copia de seguridad. A partir de la siguiente copia de seguridad, Data Replicator 3 comprobará automáticamente si ya existe algún archivo con el mismo nombre en la carpeta “latest”. Si existe y ha cambiado el contenido de los archivos, el archivo de copia de seguridad anterior de la carpeta "latest" se moverá a la carpeta "past" como versión anterior, mientras que el archivo de su equipo se copiará en la carpeta "latest", donde siempre encontrará los archivos copiados más recientes.

Observación:

El archivo al que se ha hecho referencia anteriormente debe encontrarse en la misma carpeta. De este modo, cuando use la misma cuenta para hacer una copia de seguridad de la carpeta, se habrán generado versiones anteriores.

Con la función de versiones de archivos activada, si sufre una edición inadecuada e irreversible en sus archivos locales, podrá encontrar el historial de archivos en el destino de copia de seguridad y restaurar los archivos con facilidad.

Número de versiones anteriores

Aquí puede indicar cuántas versiones anteriores de archivos de copia de seguridad desea mantener. El máximo es de 20 versiones. Al nombre de archivo de los archivos de versiones anteriores se le añadirá "v00", "v01", etc. Cuanto mayor sea el número, más reciente será la versión.

Defina un número de versiones anteriores para decidir cuántas versiones anteriores se conservarán en el destino de copia de seguridad. Siga estos pasos para continuar:

  1. Active Habilitar versiones de archivos.
  2. Defina un número de versiones anteriores y haga clic en Aplicar para guardar la configuración.

Puntos de restauración

¿Qué son los puntos de restauración?

Cada vez que se realiza una copia de seguridad, habrá una carpeta con el nombre de la hora de inicio de la copia de seguridad, que contendrá todos los archivos de esta copia de seguridad. Se trata de un punto de restauración.

Los archivos que no se hayan modificado nunca aparecerán en todos los puntos de restauración después de cada copia de seguridad. Sin embargo, en realidad sólo estará ocupado un espacio de disco para el archivo.

Número de puntos de restauración

Aquí puede indicar el número de puntos de restauración que desee conservar, con un máximo de 128. Si el número de puntos de restauración supera el valor establecido, se borrará automáticamente el más antiguo en el destino de la copia de seguridad tras la siguiente copia de seguridad.

Soporte y Limitaciones
La función Punto de restauración sólo se admite en Windows 2000 / Windows XP / Windows 2003 / Windows Vista.

Inicio automático al iniciar sesión

Puede configurar Synology Data Replicator 3 para que se inicie automáticamente cuando inicie sesión en Windows. Siga estos pasos para continuar:

  1. Active Iniciar automáticamente Data Replicator 3 al iniciar Windows.
  2. Haga clic en Aplicar.

Comprobar la consistencia de datos antes de sincronizar

Puede decidir si desea comprobar la consistencia de datos antes de que Data Replicator entre en modo de sincronización. Siga estos pasos para continuar:

  1. Active Comprobar la consistencia de datos antes de sincronizar.
  2. Haga clic en Aplicar.

Observación:

Si desactiva esta opción y cierra Data Replicator, los datos agregados, quitados o modificados podrían no sincronizarse con el servidor la próxima vez que inicie Data Replicator.

Conservar archivos idénticos en servidor de destino

Puede hacer que los archivos de la carpeta "latest" sean idénticos a los del equipo local. Siga estos pasos para continuar:

  1. Active Cuando elimine archivos locales, elimine también los archivos de copia de seguridad opuestos al mismo tiempo.
  2. Haga clic en Aplicar.

Registro

El tamaño máximo del archivo de registro es de 512 KB; si el archivo de registro supera el límite, se quitarán los 128 KB más antiguos.

  • Guardar registro
    1. Haga clic en Guardar.
    2. Elija la ruta del destino de copia de seguridad y haga clic en Guardar para finalizar.

  • Limpiar registro
    1. Haga clic en Borrar.
    2. Haga clic en Sí en el cuadro de diálogo de confirmación.
