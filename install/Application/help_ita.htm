Utilizzo di Data Replicator 3

Fare clic su una delle seguenti funzioni principali per cominciare a conoscere Data Replicator 3.

  • Backup dati PC
  • Ripristino file di backup
  • Impostazioni avanzate
  • Registro

Backup dati PC

Seguire i passi indicati di seguito per impostare il backup. è anche possibile accedere alle Impostazioni avanzate per apprendere le altre impostazioni utili alla personalizzazione del backup dei dati contenuti nel PC.

  1. Selezionare una destinazione di backup

    2. È possibile scegliere di effettuare il backup dei dati del PC su un Synology Server o in un'altra posizione. Al termine della procedura di backup, i file si troveranno nella cartella assegnata.

    1.1 Backup su un Synology Server

    1. Fare clic su Selezionare.
    2. Selezionare Synology Server e fare clic su OK.
    3. Nella finestra successiva, fare clic sul Synology Server in cui si trova la cartella condivisa per il backup. Quindi fare clic su Avanti.
    4. Immettere nome utente e password dell'utente che ha diritto di Lettura/Scrittura della cartella condivisa scelta e fare clic su Avanti.
    5. Fare clic sulla cartella condivisa che si desidera utilizzare per il backup e poi su Fine.

    1.2 Backup su un'altra posizione

    1. Fare clic su Selezionare.
    2. Selezionare Altra Posizione, fare clic su Sfoglia per individuare la cartella di destinazione per il backup.
    3. Fare clic su OK per terminare.

  2. Contrassegnare le cartelle da sottoporre a backup

  3. Scegliere una modalità di backup

    4. Data Replicator 3 consente di effettuare il backup in 3 modi diversi: Immediato, Sync, and Pianificato.

    3.1 Immediato

    Facendo clic sul pulsante Immediato, tutti i file nelle cartelle locali selezionate saranno inviati direttamente alla cartella di destinazione del backup.

    3.2 Sync

    • Effettua subito la sincronizzazione e avvia il monitoraggio

      • Facendo clic sul pulsante Sync, Data Replicator 3 effettuerà un backup completo di tutti i file nelle cartelle selezionate nella destinazione del backup. Dopo di ciò, Data Replicator 3 avvierà il monitoraggio di quelle cartelle. Via via che vengono effettuate delle modifiche, Data Replicator 3 effettua il backup dei soli file modificati.

      Nota:

      1. la modalità di monitoraggio di Data Replicator 3 può essere arrestata solo manualmente.
      2. L'intervallo temporale tra la sincronizzazione della posta si riferisce al tempo che intercorre tra il completamento di un'attività e l'inizio di quella successiva. L'attività può essere posticipata quando Data Replicator 3 sta elaborando la sincronizzazione di altri file.

    • Arresto del monitoraggio

      • Per fermare il monitoraggio, fare clic sul pulsante Stop Sync nella finestra di Data Replicator 3. Se la finestra è stata ridotta ad icona, fare doppio clic sull'icona per aprire Data Replicator 3.

    3.3 Pianifica

    • Impostare Pianificato

      • Data Replicator 3 può essere impostato per creare backup automatici ad intervalli di tempo regolari. Per fare ciò:

      1. Fare clic su Pianifica.
      2. Impostare il periodo di tempo e l'orario di inizio del Backup Pianificato e fare clic su OK.
      3. Se si desidera effettuare un Backup Pianificato quando non si è connessi, immettere nome utente e password di accesso al computer. Quindi fare clic su OK.
        Se non si desidera effettuare il Backup Pianificato quando non si è connessi, fare clic su Ignora.

      Nota:

      1. Per effettuare il backup quando non si è collegati la password utente non può essere vuota.
      2. Per gli utenti di Windows 98 e Windows ME, Backup Pianificato può essere effettuato anche durante il logout.

      Al termine delle impostazioni, è possibile uscire da Synology Data Replicator 3 o ridurre ad icona la finestra del programma. Synology Data Replicator 3 effettua la pianificazione del backup automaticamente al momento opportuno.

    • Elimina pianificazione

      • Fare clic su Elimina pianificazione per interrompere il Backup Pianificato e rimuovere tutte le relative impostazioni.

    Se si desidera effettuare il backup dei file di posta di Outlook, Outlook Express, o Windows Mail, si noti:

    • Se si utilizza Windows XP e Windows 2003, o Windows Vista,
      1. Almeno una unità sul vostro PC deve essere formattata in NTFS.
      2. I file di Outlook/Windows Mail devono trovarsi su dischi locali.
      3. Gli utenti devono accedere come amministratori.
    • Se il vostro sistema operativo non è Windows XP o 2003, chiudere Outlook prima di effettuare il backup.

      • Supporto & Limitazioni

      1. Quando si ricerca un server di destinazione, Synology Data Replicator 3 usa le porte 9999, 9998 o 9997.
      2. I file compressi e crittografati perderanno queste caratteristiche dopo il backup. Le proprietà dei file saranno cancellate.
      3. Per la lunghezza dei nomi di file, Synology Data Replicator 3 supporta fino a 240 byte.
      4. Se si cambia il nome del computer, è necessario inserire la password di accesso dell'admin del PC quando si imposta Pianifica Backup per assicurare il corretto funzionamento di Data Replicator 3. Se si è già impostato un backup pianificato, cancellare la programmazione ed usare la password di accesso dell'amministratore del PC per impostare un nuovo backup.

Ripristino file di backup

Per ripristinare i file di backup è necessario selezionare prima l'unità di destinazione e quindi scegliere uno dei seguenti metodi:

  1. Ripristino mediante esplorazione

    2. In molte situazioni, se si conosce la posizione dei file di backup è possibile usare Ripristino mediante Esplorazione per ripristinare i file:

    1. Fare clic sul pulsate Avvia in Ripristino mediante Esplorazione.
    2. Fare clic su Avanti.
    3. Fare clic per selezionare la sorgente e poi su Avanti.
    4. Scegliere se elencare i file Per versione o Per punto di ripristino.
    5. Selezionare le cartelle da ripristinare e fare clic su Avanti.
    6. Scegliere la posizione in cui i file saranno ripristinati. Al termine, fare clic su Fine.

  2. Ripristino mediante Ricerca

    3. Se non si conosce la posizione dei file di backup, o se si vuole ripristinare solo alcuni file, effettuare una delle operazioni indicate in seguito:

    1. Fare clic sul pulsante Avvia in Ripristino mediante Esplorazione.
    2. Fare clic su Avanti.
    3. Fare clic per selezionare la sorgente e poi su Avanti.
    4. Scrivere la parola chiave da ricercare nel campo "Parola chiave": campo di testo.
    5. Scegliere la cartella di backup dal menu a discesa "Backup": menu a discesa.
    6. Fare clic su Cerca.
    7. Selezionare i file e le cartelle che si desidera ripristinare nella parte a destra e fare clic su Avanti.
    8. Scegliere la posizione in cui i file saranno ripristinati. Al termine, fare clic su Fine.

    Suggerimenti per il ripristino dei file di Outlook, Outlook Express, e Windows Mail

    Per la scelta di un percorso di destinazione, scegliere tra:

    1. Percorso originale
      2.

      scegliendo questa opzione, non si ha bisogno di importare manualmente i messaggi e la rubrica, i dati sul PC saranno sostituiti dai file ripristinati.

    2. Nuovo percorso
      3.

      scegliendo questa opzione, si può ripristinare i dati in un percorso specifico ed importare manualmente i messaggi e la rubrica con Outlook/Outlook Express e Windows Mail.

Impostazioni avanzate

Nella sezione Opzioni si accede alle impostazioni avanzate di backup.

Filtro

Usare questa funzione per effettuare un backup di precisione. Escludendo i file con un dato prefisso o estensione dall'elenco di backup renderà l'operazione di backup più rapida. Procedere come segue:

  1. Fare clic su Filtra.
  2. Fare clic su Aggiungi alla destra dell'opzione "Estensione" o "Prefix".
  3. Inserire un criterio specifico, ad esempio, ".exe" come estensione o "~" come prefisso.
  4. Fare clic su OK per chiudere la finestra di dialogo e fare clic di nuovo su OK per salvare le impostazioni.

Abilita notifica e-mail

Usare questa funzione per ricevere i dettagli del backup per email al termine del Backup pianificato.

  1. Fare clic su Configurazione SMTP.
  2. Inserire le informazioni relative all'e-mail e fare clic su OK per terminare.

Abilita versioni de file

A che cosa serve la funzione versioni file?

Abilitando questa funzione si può conservare uno storico delle versioni dello stesso file.

Dopo la prima operazione di backup, tutti i file saranno archiviati nella cartella "latest" nella posizione di destinazione. Dal backup successivo, Data Replicator 3 controllerà automaticamente se esiste uno stesso file con lo stesso nome nella cartella "latest". Se c'è e il contenuto dei file è cambiato, il file precedentemente archiviato nella cartella "latest" sarà spostato nella cartella "past", mentre il file sul computer sarà archiviato nella cartella "latest" dove sempre è possibile trovare gli ultimi backup dei file.

Nota:

Il file cui si è accennato sopra deve trovarsi nella stessa cartella. Quindi, quando si usa uno stesso account per il backup della cartella, sarà generata una versione precedente.

Quando è attiva la funzione della versione del file e i file locali vengono modificati in modo improprio ed irreversibile, le versioni precedenti possono essere agevolmente trovate nella destinazione di backup e ripristinate.

Numero de versioni precedenti

È possibile impostare il numero di versioni precedenti da conservare. Il numero minimo è 20. Il nome dei file delle versioni precedenti sarà aggiunto con “v00”, “v01”, e così via. Più è alto il numero e più recente è la versione.

Impostare un numero di versioni precedenti per decidere quante versioni conservare. Procedere come segue:

  1. Selezionare Abilita versioni file.
  2. Impostare un numero di versioni precedenti e fare clic su Applica per salvare le impostazioni.

Punti di ripristino

Che cosa sono i punti di ripristino?

Ogni qual volta di effettua un backup, una cartella di backup sarà rinominata con la data e l'ora dell'avvio del backup. Questo è il punto di ripristino.

Il file che non è mai stato modificato apparirà in ogni punto di ripristino dopo ogni bakcup. Tuttavia, soltanto un file dalle dimensioni di un disco è effettivamente occupato.

Numero di punti di ripristino

È possibile impostare il numero dei punti di ripristino da conservare. Il numero massimo è 128. Quando il punto di ripristino supera il valore impostato, i punti di ripristino più vecchi saranno cancellati automaticamente dopo il backup successivo.

Supporto & Limitazioni
La funzione Punto di ripristino è supportata soltanto in Windows 2000 / Windows XP / Windows 2003 / Windows Vista.

Avvio automatico al Logon

È possibile impostare l'avvio di Synology Data Replicator 3 automaticamente all'accesso a Windows. Procedere come segue:

  1. Selezionare Avvia Data Replicator 3 con Windows.
  2. Fare clic su Applica.

Controllare la coerenza dei dati prima della sincronizzazione

È possibile decidere se controllare la coerenza dei dati prima che Data Replicator entri in modalità sincronizzazione. Procedere come segue:

  1. Selezionare Controllare la coerenza dei dati prima della sincronizzazione.
  2. Fare clic su Applica.

Nota:

se si deseleziona questa opzione e si chiude Data Replicator, i dati aggiunti, rimossi o modificati potrebbero non essere sincronizzati al server al prossimo avvio di Data Replicator.

Conserva i file identici sul server di destinazione

Si possono conservare i file nell' "latest" cartella in modo da rispecchiare quelli contenuti nel computer locale. Procedere come segue:

  1. Selezionare Quando si cancellano i file locali, cancellare nel contempo i relativi file di backup.
  2. Fare clic su Applica.

Registro

La dimensione massima del file di log è 512KB; quando il file raggiunge tale limite, i 128KB più vecchi saranno eliminati.

  • Salva Registro
    1. Fare clic su Salva.
    2. Scegliere il percorso della destinazione di backup e cliccare su Salva per terminare.

  • Pulisci Registro
    1. Fare clic su Cancella.
    2. Fare clic su Sì nella finestra di conferma.
