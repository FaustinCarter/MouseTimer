Usar o Data Replicator 3

Clique nas seguintes funções principais para conhecer melhor o Data Replicator 3.

  • Cópia de segurança dos dados do computador
  • Cópia de segurança da restauração de ficheiros
  • Definições Avançadas
  • Registo

Cópia de segurança dos dados do computador

Siga os passos abaixo indicados para configurar a sua cópia de segurança. Também poderá ir a Definições Avançadas para conhecer outras definições que o ajudarão a personalizar a cópia de segurança de dados do seu computador.

  1. Seleccione um destino da cópia de segurança

    2. Pode optar por fazer a cópia de segurança dos dados do seu computador num servidor Synology ou noutra localização. Depois de a cópia de segurança ser realizada com sucesso, poderá encontrar os ficheiros de cópia de segurança na sua pasta atribuída.

    1.1 Fazer a cópia de segurança num servidor Synology

    1. Clique em Seleccionar.
    2. Seleccione Servidor Synology e clique em OK.
    3. Na próxima janela, clique no servidor Synology onde está localizada a pasta partilhada para a cópia de segurança. De seguida, clique em Seguinte.
    4. Introduza o nome de utilizador e a palavra-passe que pode Ler/Escrever na pasta partilhada que acabou de escolher, e clique em Seguinte.
    5. Clique na pasta partilhada que pretende usar para a cópia de segurança e clique em Terminar.

    1.2 Fazer cópia de segurança noutra localização

    1. Clique em Seleccionar.
    2. Seleccione Outra localização e clique em Pesquisar para localizar o destino da cópia de segurança.
    3. Clique em OK para terminar.

  2. Coloque um visto nas pastas de que pretende fazer a cópia de segurança

  3. Escolha um modo de cópia de segurança

    4. O Data Replicator 3 fornece 3 modos de cópia de segurança para sua maior comodidade, que são: Imediata, Sincronização e Agendada.

    3.1 Imediata

         Clicando no botão Imediata, será feita uma cópia de segurança imediata de todos os ficheiros nas pastas locais seleccionadas para o destino da cópia de segurança.

    3.2 Sincronização

    • Sincronizar os dados agora e começar a monitorização

      • Clique no botão Sincronização; o Data Replicator 3 fará uma cópia de segurança completa de todos os ficheiros das pastas seleccionadas para o destino da cópia de segurança. Depois disso, o Data Replicator 3 começará a monitorização dessas mesmas pastas. À medida que fizer alterações, o Data Replicator 3 fará apenas cópias de segurança dos ficheiros que forem modificados.

      Nota:

      1. Depois de o Data Replicator 3 entrar no modo de monitorização, continuará a trabalhar até que o pare.
      2. O tempo de intervalo entre a sincronização e-mail diz respeito ao tempo desde que a sincronização termina até ao início da próxima sincronização. A tarefa pode ser adiada quando o Data Replicator 3 estiver a processar a sincronização de outros ficheiros.

    • Parar a monitorização

      • Para parar a monitorização, clique no botão Parar a sincronização na janela do programa Data Replicator 3. Se a janela do programa foi minimizada para a tabuleiro do sistema, clique duas vezes no ícone do Data Replicator 3 para a abrir.

    3.3 Agendar

    • Configurar horário

      • O Data Replicator 3 pode ser configurado para fazer cópias de segurança automáticas regularmente no horário especificado. Para isso:

      1. Clique em Agendar.
      2. Configure o período da cópia de segurança e a hora de início para a cópia de segurança agendada e clique em OK.
      3. Se pretende realizar a cópia de segurança durante o encerramento de sessão, introduza o nome de utilizador e a palavra-passe que usa para iniciar a sessão neste computador. De seguida, clique em OK.
        Se não pretende realizar a cópia de segurança agendada durante o fim de sessão, clique em Ignorar.

      Nota:

      1. Para fazer a cópia de segurança durante o fecho de sessão, o utilizador e a palavra-passe não podem estar em branco.
      2. Para utilizadores com Windows 98 ou Windows ME, a cópia de segurança agendada pode ser realizada durante o fecho de sessão por defeito.

      Depois de configurar o horário, pode sair do Synology Data Replicator 3 ou minimizar a janela do programa para o tabuleiro do sistema. O Synology Data Replicator 3 irá realizar a cópia de segurança agendada automaticamente no horário indicado.

    • Apagar o agendamento

      • Clique em Apagar o agendamento para parar a cópia de segurança agendada e remover as configurações de agendamento.

    Quando pretender fazer a cópia de segurança dos ficheiros do Outlook, Outlook Express ou do Windows Mail, tenha em conta que:

    • Se estiver a executar o Windows XP, Windows 2003 ou o Windows Vista,
      1. Pelo menos uma das drives do seu computador deve estar no formato NTFS.
      2. Os ficheiros do Outlook/Windows Mail devem estar nas drives locais.
      3. Tem de iniciar a sessão como administrador do computador.
    • Se estiver a executar um sistema operativo sem ser Windows XP e o Windows 2003, feche o Outlook antes de fazer a cópia de segurança do mesmo.

      • Suporte e Limitação

      1. Ao pesquisar o servidor alvo, o Synology Data Replicator 3 usará a porta 9999, ou 9998, ou 9997.
      2. Os ficheiros comprimidos e os ficheiros encriptados tornar-se-ão ficheiros normais depois da cópia de segurança. A propriedade do ficheiro será apagada.
      3. Para o comprimento do nome do ficheiro, o Synology Data Replicator 3 permite até 240 bytes.
      4. Se mudar o nome do computador do seu PC, deve introduzir a palavra-passe de início de sessão do administrador do seu computador ao configurar a cópia de segurança agendada para garantir que o Data Replicator 3 a realiza correctamente. Se já configurou um horário para a cópia de segurança, apague esse horário e use a palavra-passe de início de sessão do administrador do seu computador para definir um novo.

Cópia de segurança da restauração de ficheiros

Para restaurar ficheiros de cópia de segurança, deve seleccionar um alvo de cópia de segurança primeiro, e depois escolher um dos seguintes métodos para restaurar os seus ficheiros de cópia de segurança:

  1. Restaurar por navegação

    2. Na maior parte das situações, se tiver a certeza da localização dos ficheiros de cópia de segurança, pode usar Restaurar por navegação para restaurar os seus ficheiros:

    1. Clique no botão Iniciar debaixo de Restaurar por navegação.
    2. Clique em Seguinte.
    3. Clique para seleccionar a fonte de restauro e clique em Seguinte.
    4. Escolha se pretende mostrar os ficheiros por versão ou por ponto de restauro.
    5. Coloque um visto nos ficheiros que pretende restaurar, e clique em Seguinte.
    6. Escolha a localização para onde os ficheiros de cópia de segurança serão restaurados. De seguida, clique em Terminar.

  2. Restaurar por pesquisa

    3. Se não tiver certeza de onde estão os ficheiros de cópia de segurança, ou se só pretende restaurar ficheiros com determinada palavra-chave, siga os seguintes passos para restaurar:

    1. Clique no botão Iniciar debaixo do restaurar por pesquisa.
    2. Clique em Seguinte.
    3. Clique para seleccionar a fonte de restauro e clique em Seguinte.
    4. Digite a palavra-chave que pretende pesquisar no campo de texto "Palavra-chave:" .
    5. Escolha a pasta de cópia de segurança no menu pendente "Cópias de segurança:" .
    6. Clique em Procurar.
    7. Seleccione os ficheiros e pastas que pretende restaurar na moldura do lado direito e clique em Seguinte.
    8. Escolha a localização para onde os ficheiros de cópia de segurança serão restaurados. De seguida, clique em Terminar.

    Dica sobre a restauração de ficheiros Outlook, Outlook Express e Windows Mail

         Ao escolher o caminho de destino, pode optar por:

    1. Caminho original
      2.

      Ao escolher esta opção, não precisa de importar manualmente o e-mail e a lista de endereços; os dados do computador serão substituídos pelos ficheiros restaurados.

    2. Novo caminho
      3.

      Ao escolher esta opção, pode restaurar os dados para um caminho específico e importar manualmente o mail e o livro de endereços com o Outlook/Outlook Express/Windows Mail.

Definições Avançadas

Na secção desta Opção, pode configurar as definições avançadas da cópia de segurança como se mostra a seguir.

Filtro

Use esta função para fazer cópias de segurança de ficheiros com precisão. Excluir ficheiros com determinados prefixos e extensões da lista de cópia de segurança ajuda-lo-á a configurar a cópia de segurança. Siga estes passos para continuar:

  1. Clique em Filtro.
  2. Clique em Adicionar no lado direito da opção "Extensão" ou "Prefix".
  3. Introduza o critério específico - por exemplo, "exe" para extensão ou "~" para prefixo.
  4. Clique em OK para fechar a caixa de diálogo de configuração e clique em OK novamente para guardar as configurações.

Activar notificação por e-mail

Use esta função para receber os dados da cópia de segurança por e-mail depois da cópia de segurança agendada terminar.

  1. Clique em Configurar SMTP.
  2. Introduza a informação de e-mail válida e clique em OK para terminar.

Activar as versões dos ficheiros

Para que serve a função de versões dos ficheiros?

Activar as versões dos ficheiros pode ajudá-lo a guardar o histórico das edições do mesmo ficheiro.

Após a conclusão da primeira cópia de segurança, todos os ficheiros com cópia de segurança serão guardados na pasta "latest" no destino da cópia de segurança. Na próxima cópia de segurança, o Data Replicator 3 verificará automaticamente se já existe algum ficheiro com o mesmo nome na pasta "latest". Se houver e se o conteúdo dos ficheiros foi modificado, a anterior cópia de segurança do ficheiro na pasta "latest" será então movida para a pasta "past", como uma versão anterior. No seu computador será feita uma cópia de segurança do ficheiro na pasta "latest", onde poderá sempre encontrar todas as cópias de segurança recentes.

Nota:

O ficheiro referido em cima deve estar na mesma pasta. Portanto, quando está a usar a mesma conta para fazer uma cópia de segurança da pasta, será gerada uma versão passada.

Quando a função de versão de ficheiro está activa, se houver edição imprópria ou irreversível nos seus ficheiros locais, pode encontrar o histórico dos ficheiros no destino de cópia de segurança e restaurar os ficheiros facilmente.

Número das versões anteriores

Pode definir aqui quantas versões anteriores de cópia de segurança dos ficheiros pretende guardar; 20 versões é o máximo. Ao nome do ficheiro dos ficheiros de versão passado será adicionado "v00", "v01" e assim sucessivamente. Quanto maior for o número, mais recente é a versão.

Configurar um número de versões anteriores para decidir quantas versões anteriores serão guardadas no destino de cópia de segurança. Siga estes passos para continuar:

  1. Coloque um visto em Activar versões de ficheiro.
  2. Defina um número para as versões anteriores e clique em Aplicar para guardar as configurações.

Pontos de restauro

O que são pontos de restauro?

Sempre que uma cópia de segurança é realizada, haverá uma pasta cujo nome será a hora de início da cópia de segurança que contém todos os ficheiros dessa cópia de segurança. Esse será o ponto de restauro.

O ficheiro que nunca foi modificado aparecerá em todos os pontos de restauro depois de cada cópia de segurança. No entanto, apenas um tamanho do disco do ficheiro será ocupado na realidade.

Número de pontos de restauro

Pode definir aqui quantos pontos de restauro pretende guardar; o máximo são 128 pontos de restauro. Quando o número de pontos de restauro exceder o valor definido, os pontos de restauro mais antigos no destino da cópia de segurança serão apagados automaticamente na seguinte cópia de segurança.

Suporte e Limitação
A função dos pontos de restauro é apenas suportada no Windows 2000 / Windows XP / Windows 2003 / Windows Vista.

Inicialização automática no arranque

Pode definir para iniciar o Synology Data Replicator 3 automaticamente quando inicia a sessão no Windows. Siga estes passos para continuar:

  1. Coloque um visto em Inicialização automática do Data Replicator 3 no arranque do Windows.
  2. Clique em Aplicar.

Verificar a consistência dos dados antes de sincronizar

Pode decidir se deve ser verificada a consistência dos dados antes do Data Replicator entrar no modo de sincronização. Siga estes passos para continuar:

  1. Assinale a caixa de verificação Verificar a consistência dos dados antes de começar a sincronizar.
  2. Clique em Aplicar.

Nota:

Se desmarcar esta opção e fechar o Data Replicator, os dados adicionados, removidos ou modificados podem não ser sincronizados quando iniciar o Data Replicator da próxima vez.

Manter os ficheiros idênticos no servidor de destino

Se puder definir guardar os ficheiros na pasta "latest" para serem idênticos aos que estão no seu computador local. Siga estes passos para continuar:

  1. Coloque um visto em Quando os ficheiros locais são apagados, apagar os ficheiros de cópia de segurança correspondentes ao mesmo tempo.
  2. Clique em Aplicar.

Registo

O máximo do ficheiro de registo é 512KB; quando o ficheiro de registo atinge o limite, os 128KB mais antigos serão removidos.

  • Guardar registo
    1. Clique em Guardar .
    2. Escolha o caminho do destino da cópia de segurança e clique em Guardar para terminar.

  • Limpar registo
    1. Clique em Limpar.
    2. Clique em Sim na caixa de diálogo de confirmação .
