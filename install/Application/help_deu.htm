Data Replicator 3 verwenden

Klicken Sie die folgenden Hauptfunktionen an, um sich mit dem Data Replicator 3 vertraut zu machen.

  • PC-Datensicherung
  • Sicherungsdateien wieder herstellen
  • Erweiterte Einstellungen
  • Protokoll

PC-Datensicherung

Bitte befolgen Sie die unten aufgeführten Schritte, um Ihre Datensicherung einzurichten: Sie können auch Erweiterte Einstellungen anklicken, um weitere Einstellungen zu erlernen, mit denen Sie Ihre PC-Datensicherung persönlich gestalten können.

  1. Einen Zielpfad wählen

    2. Sie können Ihre Daten auf einem Synology Server oder an einem anderen Ort speichern. Nachdem die Datensicherung erfolgreich durchgeführt wurde, können Sie gesicherten Dateien wieder aus dem von Ihnen zugewiesenen Ordner aufrufen.

    1.1 Auf einem Synology Server sichern

    1. Klicken Sie Auswählen an.
    2. Wählen Sie Synology Server und klicken Sie OK.
    3. Im nächsten Fenster klicken Sie den Synology Server an, auf dem sich der gemeinsam genutzte Ordner zur Datensicherung befindet. Klicken Sie anschließend auf Weiter.
    4. Geben Sie den Benutzernamen und das Kennwort für den Schreib-/Lesezugriff auf den von Ihnen ausgewählten gemeinsam genutzten Order ein und klicken Sie auf Weiter.
    5. Klicken Sie den gemeinsamen Ordner an, den Sie zur Datensicherung verwenden möchten und klicken Sie Fertig stellen.

    1.2 Datensicherung an einem anderen Speicherort

    1. Klicken Sie Auswählen an.
    2. Wählen Sie Anderer Speicherort und klicken Sie die Schaltfläche Durchsuchen an.
    3. Klicken Sie auf OK, um den Vorgang zu beenden.

  2. Kreuzen Sie die Ordner an, die Sie sichern möchten

  3. Wählen Sie einen Datensicherungsmodus

    4. Data Replicator 3 bietet Ihnen 3 Datensicherungsmodi zur Auswahl an. Diese sind: Sofort, Synchronisierung und Zeitplan.

    3.1 Sofort

    Wenn Sie die Schaltfläche Sofort anklicken, werden alle Dateien im ausgewählten lokalen Ordner sofort im Zielpfad gespeichert.

    3.2 Sync

    • Synchronisieren Sie Ihre Daten jetzt und beginnen Sie mit der Überwachung

      • Wenn Sie die Schaltfläche Synchronisation anklicken, sichert Data Replicator 3 zunächst alle Dateien im ausgewählten lokalen Ordner im Zielpfad. Anschließend beginnt Data Replicator 3 mit der Überwachung Ihrer/s lokalen Ordner/s. Wenn Sie Änderungen vornehmen, sichert Data Replicator 3 nur jene Dateien, die Sie geändert haben.

      Anmerkung:

      1. Nachdem Data Replicator 3 in den Überwachungsmodus übergeht, arbeitet er weiter, bis Sie den Vorgang abbrechen.
      2. Der Zeitintervall zwischen Mail-Synchronisationen bezieht sich auf die Zeitdauer zwischen dem Abschluss einer Synchronisation und dem Start der nächsten. Die Aufgabe kann verschoben werden, während Data Replicator 3 andere Dateien synchronisiert.

    • Überwachung stoppen

      • Klicken Sie die Schaltfläche Synchronisation stoppen im Programmfenster von Data Replicator 3. Wenn das Programm auf die Taskleiste minimiert ist, doppelklicken Sie Data Replicator 3 an, um es zu öffnen.

    3.3 Plan

    • Zeitplan einstellen

      • Data Replicator 3 kann so eingestellt werden, dass er regelmäßig zur vorgegebenen Zeit automatische Datensicherungen vornimmt. So verfahren Sie:

      1. Klicken Sie Zeitplan.
      2. Stellen Sie die Sicherungszeitdauer und die Startzeit für die eingestellte Datensicherung ein und klicken Sie OK.
      3. Wenn Sie eine eingestellte Dateisicherung während des Abmeldens durchführen möchten, geben Sie bitte den Benutzernamen und das Kennwort ein, die Sie zum Anmelden auf diesem PC verwendet haben. Klicken Sie dann auf OK.
        Wenn Sie beim Abmelden keine eingestellte Datensicherung durchführen möchten, klicken Sie bitte Überspringen.

      Anmerkung:

      1. Wenn Sie beim Abmelden Daten sichern möchten, darf das Kennwortfeld nicht leer bleiben.
      2. Bei Windows 98 und Windows ME Anwendern kann die eingestellte Datensicherung während der Abmeldung durchgeführt werden.

      Wenn der Zeitplan eingestellt ist, können Sie Synology Data Replicator 3 verlassen oder das Programm in die Taskleiste minimieren. Synology Data Replicator 3 führt die eingestellte Datensicherung automatisch durch, wenn die Zeit abgelaufen ist.

    • Zeitplan löschen

      • Klicken Sie Zeitplan löschen, um die eingestellte Datensicherung zu stoppen und die Einstellungen zu entfernen.

    Wenn Sie Outlook-, Outlook Express- oder Windows Mail-Dateien sichern möchten, beachten Sie bitte folgendes:

    • Wenn Sie Windows XP, Windows 2003 oder Windows Vista haben,
      1. sollte mindestens eines Ihrer Laufwerke auf Ihrem PC im NTFS-Format sein.
      2. Outlook-/Windows Mail-Dateien sollten auf lokalen Laufwerken gespeichert sein.
      3. Sie müssen sich als PC Administrator anmelden.
    • Wenn Sie ein Betriebssystem außer Windows XP und Windows 2003 haben, schließen Sie Outlook bitte vor der Sicherung.

      • Unterstützung und Haftungsbeschränkungen

      1. Beim Suchen nach dem Zielserver verwendet der Synology Data Replicator 3 den Port 9999 oder 9998 oder 9997.
      2. Komprimierte und verschlüsselte Dateien werden nach dem Backup zu gewöhnlichen Dateien. Die Dateieigenschaften werden gelöscht.
      3. Synology Data Replicator 3 unterstützt bis zu 240 Bytes für den Dateinamen.
      4. Wenn Sie den Computernamen Ihres PCs ändern, müssen Sie das Administrator-Anmeldekennwort Ihres PCs eingeben, wenn Sie eine geplante Datensicherung einstellen, um sicherzustellen, dass der Data Replicator 3 korrekt läuft. Wenn Sie bereits eine geplante Datensicherung eingegeben haben, müssen Sie diese löschen und das Administrator-Anmeldekennwort Ihres PCs verwenden, um eine neue Sicherung einzustellen.

Sicherungsdateien wieder herstellen

Um gesicherte Dateien wiederherzustellen, müssen Sie zuerst einen Datensicherungs-Zielpfad festlegen und dann eine der folgenden Methoden wählen, um Ihre gesicherten Dateien wiederherzustellen:

  1. Wiederherstellen mittels Durchsuchen

    2. In den meisten Fällen können Sie die Funktion Wiederherstellen mittels Durchsuchen verwenden, um Ihre Dateien wiederherzustellen:

    1. Klicken Sie die Schaltfläche Start unter Wiederherstellen mittels Durchsuchen.
    2. Klicken Sie auf Weiter.
    3. Anklicken, um die Wiederherstellungsquelle zu wählen und anschließend Weiter anklicken.
    4. Wählen Sie, ob Sie Dateien Nach Version oder Durch Wiederherstellungspunkt auflisten möchten.
    5. Wählen Sie die wiederherzustellenden Ordner aus und klicken Sie Weiter.
    6. Wählen Sie den Zielpfad, in dem die gesicherten Dateien wieder hergestellt werden sollen. Klicken Sie anschließend Fertig stellen.

  2. Wiederherstellen mittels Suchen

    3. Wenn Sie sich über den Ort der wiederherzustellenden Dateien nicht sicher sind, oder nur Dateien mit bestimmten Schlüsselwörtern wiederherstellen möchten, befolgen Sie bitte die folgenden Schritte zur Wiederherstellung:

    1. Klicken Sie die Schaltfläche Start unter Wiederherstellen mittels Suchen.
    2. Klicken Sie auf Weiter.
    3. Anklicken, um die Wiederherstellungsquelle zu wählen und anschließend Weiter anklicken.
    4. Schreiben Sie das Schlüsselwort, dass Sie suchen möchten in "Schlüsselwort:" Textfeld.
    5. Wählen Sie den Sicherungsordner aus „Sicherungen“: Drop-Down Menü.
    6. Klicken Sie Suche.
    7. Wählen Sie die Dateien und Ordner, die Sie wiederherstellen möchten im rechten Rahmen und klicken Sie Weiter.
    8. Wählen Sie den Zielpfad, in dem die gesicherten Dateien wieder hergestellt werden sollen. Klicken Sie anschließend Fertig stellen.

    Tipps zur Wiederherstellung von Outlook-, Outlook Express- und Windows Mail-Dateien

    Wenn Sie einen Zielpfad wählen, können Sie wie folgt auswählen:

    1. Originalpfad
      2.

      Wenn Sie diese Option wählen, müssen Sie Mail und Adressbuch nicht manuell importieren, PC-Daten werden durch die wiederhergestellten Dateien ersetzt.

    2. Neuer Pfad
      3.

      Wenn Sie diese Option wählen, können Sie sie die Daten auf einem festgelegten Pfad wiederherstellen und müssen Mail und Adressbuch mit Outlook und Outlook Express/Windows Mail manuell importieren.

Erweiterte Einstellungen

Unter Optionen können Sie wie folgt erweiterte Einstellungen zur Datensicherung vornehmen.

Filter

Verwenden Sie diese Funktion, um die Dateien präzise zu sichern. Das Ausschließen von Dateien mit bestimmten Präfixen oder Erweiterungen von der Datensicherungsliste kann Ihnen dabei helfen, den Vorgang zu beschleunigen. Befolgen Sie die Schritte, um fortzufahren:

  1. Klicken Sie auf Filter.
  2. Klicken Sie rechts in der "Erweiterung" oder "Prefix" Option auf Hinzufügen.
  3. Geben Sie festgelegtes Kriterium, zum Beispiel ".exe" für Erweiterung oder "~" für Präfix ein.
  4. Klicken Sie OK, um das Einrichtungsdialogfenster zu schließen und nochmals OK, um die Einstellungen zu speichern.

E-Mail-Benachrichtigungen aktivieren

Verwenden Sie diese Funktion, um Benachrichtigungen per E-Mail zu erhalten, nachdem die Datensicherung abgeschlossen wurde.

  1. Klicken Sie SMTP-Einrichtung.
  2. Geben Sie eine gültige E-Mailadresse ein und klicken Sie OK, um abzuschließen.

Dateiversionen aktivieren

Wozu dient die Funktion Dateiversionen?

Die Aktivierung der Dateiversionsfunktion kann Ihnen dabei helfen, die History-Ausgaben der gleichen Datei beizubehalten.

Nachdem die erste Datensicherung abgeschlossen wurde, werden alle gesicherten Dateien im „latest” Ordner im Zielpfad gespeichert. Ab der nächsten Datensicherung prüft Data Replicator 3 automatisch, ob eine Datei mit dem gleichen Dateinamen bereits im „latest” Ordner existiert. Wenn der Inhalt der Datei allerdings anders ist, wird die zuvor gesicherte Datei aus dem „latest” Ordner als Datei mit älterer Version in den “past” Ordner verschoben, während die revidierte Datei auf Ihrem PC im „latest” Ordner gesichert wird, wo Sie stets die neuesten gesicherten Dateien finden können.

Anmerkung:

Die o.g. Datei muss sich im gleichen Ordner befinden. Wenn Sie also das gleiche Konto verwenden, um den Ordner zu sichern, wird eine vergangene Version generiert.

Wenn die Dateiversionsfunktion aktiviert ist, und Sie fehlerhafte und nicht korrigierbare Bearbeitungen an Ihren aktuellen Dateien vorgenommen haben, können Sie die History-Dateien im Dateisicherungsordner finden und ganz einfach wiederherstellen.

Anzahl der vergangenen Versionen

Hier können Sie einstellen, wie viele der vergangenen gesicherten Dateiversionen Sie behalten möchten; es sind max. 20 Versionen. Der Dateiname von vergangenen Dateien wird dann mit “v00”, “v01”, usw. versehen. Je größer die Versionsnummer, desto neuer ist sie.

Stellen Sie eine Anzahl von vergangenen Versionen ein, um festzulegen, wie viele vorige Versionen im Zielpfad gespeichert bleiben. Befolgen Sie die Schritte, um fortzufahren:

  1. Wählen Sie Dateiversionen aktivieren.
  2. Legen Sie eine Anzahl von vergangenen Versionen fest und klicken Sie Anwenden, um die Einstellungen zu speichern.

Wiederherstellungpunkte

Was versteht man unter Wiederherstellungspunkten?

Jedes Mal, wenn eine Datensicherung abgeschlossen wurde, wird ein Ordner erstellt, der den Namen der Datensicherungs-Startzeit trägt und alle Dateien dieser Datensicherung enthält. Dies dient dann als Wiederherstellungspunkt.

Die nie veränderten Dateien erscheinen bei allen Datensicherungen als Wiederherstellungspunkt. Allerdings ist in Wirklichkeit nur eine Dateidiskgröße besetzt.

Anzahl der Wiederherstellungspunkte

Hier können Sie einstellen, wie viele Wiederherstellungspunkte Sie behalten möchten; es sind max. 128 Punkte. Wenn die Anzahl der Wiederherstellungspunkte den eingestellten Wert überschreitet, wird der älteste Wiederherstellungspunkt im Zielpfad nach der nächsten Datensicherung automatisch gelöscht.

Unterstützung und Haftungsbeschränkungen
Die Wiederherstellungspunkt-Funktion wird nur von Windows 2000 / Windows XP / Windows 2003 / Windows Vista unterstützt.

Automatischer Start beim Anmelden

Sie können einstellen, dass Synology Data Replicator 3 automatisch gestartet wird, wenn Sie sich bei Windows anmelden. Befolgen Sie die Schritte, um fortzufahren:

  1. Aktivieren Sie das Kontrollkästchen Data Replicator 3 beim Hochfahren von Windows automatisch starten.
  2. Klicken Sie auf Anwenden.

Datenkonsistenz vor Synchronisierung prüfen

Sie können entscheiden, ob Sie die Datenkonsistenz überprüfen möchten, ehe Data Replicator in den Synchronisationsmodus wechselt. Befolgen Sie die Schritte, um fortzufahren:

  1. Aktivieren Sie das Kontrollkästchen Datenkonsistenz vor Synchronisierung prüfen.
  2. Klicken Sie auf Anwenden.

Anmerkung:

Wenn Sie diese Option deaktivieren und den Data Replicator schließen, werden hinzugefügte, entfernte oder geänderte Daten unter Umständen nicht mit dem Server synchronisiert, wenn Data Replicator das nächste Mal gestartet wird.

Identische Dateien auf dem Zielserver speichern

Bei einer Datensicherung können Sie Ihre Dateien im „latest" Ordner speichern, damit diese identisch zu jenen auf Ihrem Computer sind. Befolgen Sie die Schritte, um fortzufahren:

  1. Das Kontrollkästchen Beim Entfernen von lokalen Dateien gleichzeitig auch deren Pendants löschen aktivieren.
  2. Klicken Sie auf Anwenden.

Protokoll

Die maximale Protokolldatei beträgt 512KB; wenn die Protokolldatei das Limit erreicht, werden die ältesten 128KB gelöscht.

  • Protokoll speichern
    1. Klicken Sie auf Speichern.
    2. Wählen Sie den Zielpfad und klicken Sie Speichern an, um den Vorgang abzuschließen.

  • Protokoll löschen
    1. Klicken Sie auf Löschen.
    2. Klicken Sie Ja im Bestätigungsdialog
